## 📊 **一、交叉引用类型 / Cross-Reference Types**

### 1.1 概念引用

**引用类型**: 文档内概念引用

**示例**:
```markdown

```

### 1.2 理论关系引用

**引用类型**: 理论间关系引用

**示例**:
```markdown
相关理论：[图的流理论](05-高级理论-图的流理论-深度改进版-2025.md)
```

### 1.3 跨模块引用

**引用类型**: 跨模块概念引用

**示例**:
