## 🗺️ **二、思维表征工具类型定义 / Mind Representation Tool Types**

### 2.1 思维导图 (Mind Maps)

**用途**: 展示主题的知识结构和层次关系

**示例结构**:

```mermaid
mindmap
  root((主题名称))
    一级概念
      二级概念
        三级概念
          详细说明
```

**创建标准**:

- 清晰的主题层次
- 完整的知识覆盖
- 合理的概念分组
- 直观的可视化

### 2.2 概念多维矩阵 (Multi-dimensional Concept Matrices)

**用途**: 对比概念的定义、关系、属性、解释、论证

**矩阵类型**:

1. **定义矩阵**: 概念定义对比
2. **关系矩阵**: 概念关系对比
3. **属性矩阵**: 属性特征对比
4. **解释矩阵**: 解释方式对比
5. **论证矩阵**: 论证方法对比

**示例结构**:

| 维度 | 概念A | 概念B | 概念C | 关系 |
|------|-------|-------|-------|------|
| 定义 | ... | ... | ... | ... |
| 属性 | ... | ... | ... | ... |
| 关系 | ... | ... | ... | ... |

### 2.3 决策树图 (Decision Trees)

**用途**: 算法选择、方法选择、场景决策

**结构**:

```mermaid
flowchart TD
    Start[决策起点] --> Condition1{条件1?}
    Condition1 -->|是| Option1[选项1]
    Condition1 -->|否| Condition2{条件2?}
    Condition2 -->|是| Option2[选项2]
    Condition2 -->|否| Option3[选项3]
```

### 2.4 证明树图 (Proof Trees)

**用途**: 展示定理证明的逻辑结构

**结构**:

```mermaid
graph TD
    Theorem[定理] --> Lemma1[引理1]
    Theorem --> Lemma2[引理2]
    Lemma1 --> Fact1[事实1]
    Lemma1 --> Fact2[事实2]
    Lemma2 --> Fact3[事实3]
```

### 2.5 控制执行数据流图 (Control Flow & Data Flow Diagrams)

**用途**: 算法执行流程、系统控制流程

**类型**:

1. **控制流图**: 算法控制流程
2. **数据流图**: 数据流转过程
3. **混合流图**: 控制流和数据流结合

**结构**:

```mermaid
flowchart LR
    Input[输入] --> Process1[处理1]
    Process1 --> Decision{判断}
    Decision -->|条件1| Process2[处理2]
    Decision -->|条件2| Process3[处理3]
    Process2 --> Output[输出]
    Process3 --> Output
```

### 2.6 论证思维图 (Argumentation Maps)

**用途**: 展示主题的论证脉络和逻辑结构

**结构**:

```mermaid
graph TD
    Claim[主要论点] --> Premise1[前提1]
    Claim --> Premise2[前提2]
    Premise1 --> Evidence1[证据1]
    Premise2 --> Evidence2[证据2]
    Claim --> Conclusion[结论]
```
