## 🎯 **一、导航说明 / Navigation Guide**

### 1.1 文档组织

所有思维表征工具集合按以下方式组织：

```
模块目录/
├── 思维表征工具集合-主题名-完整版-2025.md  # 完整工具集合
└── 各主题文档.md  # 原始主题文档（可能包含部分工具）
```

### 1.2 工具类型说明

每个工具集合包含以下类型的工具：

1. **思维导图**: 展示主题的知识结构
2. **概念多维矩阵**: 对比概念的定义、关系、属性等
3. **决策树图**: 提供算法和方法选择的决策指导
4. **证明树图**: 展示重要定理的证明逻辑结构
5. **控制执行数据流图**: 展示算法和系统的执行流程
6. **论证思维图**: 展示主题的论证脉络和逻辑结构

## 📋 **六、使用指南 / Usage Guide**

### 6.1 如何查找工具

1. **按主题查找**: 直接查看对应模块的工具集合文档
2. **按类型查找**: 查看工具集合中的对应章节
3. **按应用查找**: 使用决策树和论证图

### 6.2 如何使用工具

1. **学习理解**: 使用思维导图和概念矩阵理解知识结构
2. **算法选择**: 使用决策树选择合适算法
3. **理论证明**: 使用证明树理解证明逻辑
4. **算法实现**: 使用数据流图指导实现
5. **理论论证**: 使用论证图理解理论体系
