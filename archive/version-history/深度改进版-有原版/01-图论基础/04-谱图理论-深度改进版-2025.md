## 🎯 **1. 谱图理论的多种等价定义 / Multiple Equivalent Definitions**

### 1.1 矩阵定义

**定义 1.1.1** (图的谱 - 矩阵定义)

图 $G = (V, E)$ 的**谱**是邻接矩阵 $A$ 或拉普拉斯矩阵 $L = D - A$ 的特征值集合。

**邻接矩阵谱**：
$$\sigma(A) = \{\lambda_1, \lambda_2, \ldots, \lambda_n\}$$

**拉普拉斯矩阵谱**：
$$\sigma(L) = \{\mu_1, \mu_2, \ldots, \mu_n\}$$

其中 $\mu_1 = 0 \leq \mu_2 \leq \ldots \leq \mu_n$。

### 1.2 算子定义

**定义 1.1.2** (图的谱 - 算子定义)

图的谱是图上的线性算子的特征值。

**拉普拉斯算子**：
$$L: f \mapsto (Lf)(v) = \sum_{u \sim v} (f(v) - f(u))$$

### 1.3 几何定义

**定义 1.1.3** (图的谱 - 几何定义)

图的谱反映了图的几何结构，如连通性、展开性等。
