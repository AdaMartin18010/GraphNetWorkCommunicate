## 🗺️ **一、图论算法对比矩阵 / Graph Algorithm Comparison Matrices**

### 1.1 最短路径算法对比矩阵

| 维度 | Dijkstra | Floyd-Warshall | Bellman-Ford | A*算法 | 适用场景 |
|------|----------|---------------|--------------|--------|---------|
| **时间复杂度** | O(V²) 或 O((V+E)log V) | O(V³) | O(VE) | O(b^d) | - |
| **空间复杂度** | O(V) | O(V²) | O(V) | O(b^d) | - |
| **适用图类型** | 非负权重有向/无向图 | 任意权重有向图 | 任意权重有向图 | 加权图（需启发式） | - |
| **负权边支持** | ❌ | ✅ | ✅ | 取决于启发式 | - |
| **负环检测** | ❌ | ✅ | ✅ | ❌ | - |
| **单源/全对** | 单源 | 全对 | 单源 | 单源 | - |
| **实现难度** | 中等 | 简单 | 中等 | 较高 | - |
| **实际性能** | 优秀 | 中等 | 较差 | 优秀 | - |
| **推荐场景** | 单源最短路径（非负权重） | 全对最短路径 | 负权边图、负环检测 | 路径规划、游戏AI | - |
| **图规模限制** | 大规模（优化后） | 中小规模 | 中等规模 | 中等规模 | - |
| **数据结构** | 优先队列 | 邻接矩阵 | 边列表 | 优先队列+启发式 | - |

**选择决策**:

- 单源+非负权重 → Dijkstra
- 全对最短路径 → Floyd-Warshall
- 负权边/负环检测 → Bellman-Ford
- 路径规划（有启发式） → A*

### 1.2 最小生成树算法对比矩阵

| 维度 | Prim算法 | Kruskal算法 | Borůvka算法 | 适用场景 |
|------|----------|------------|-------------|---------|
| **时间复杂度** | O(V²) 或 O((V+E)log V) | O(E log E) | O(E log V) | - |
| **空间复杂度** | O(V) | O(V+E) | O(V+E) | - |
| **适用图类型** | 连通加权无向图 | 连通加权无向图 | 连通加权无向图 | - |
| **实现方式** | 贪心（顶点扩展） | 贪心（边排序） | 贪心（分治） | - |
| **数据结构** | 优先队列 | 并查集+排序 | 并查集 | - |
| **实现难度** | 中等 | 简单 | 中等 | - |
| **实际性能** | 密集图优秀 | 稀疏图优秀 | 并行友好 | - |
| **推荐场景** | 密集图（边数多） | 稀疏图（边数少） | 并行计算、大规模图 | - |

- 密集图（边数多） → Prim
- 稀疏图（边数少） → Kruskal
- 并行/大规模 → Borůvka
