## 🎯 **一、交叉引用类型 / Part 1: Cross-Reference Types**

### 1.1 概念引用

**定义**: 文档内和跨文档的概念引用

**引用格式**:

```markdown
参见：[概念名称](文档路径#锚点) - [简短说明]
```

**示例**:

- 文档内引用: `参见：[图的连通性](#连通性)`
- 跨文档引用: `参见：[图的连通性](02-连通性.md#连通性)`
- 跨模块引用: `参见：[网络拓扑](../02-网络拓扑/01-拓扑结构.md#网络拓扑)`

### 1.2 理论关系引用

**定义**: 理论间的关系引用

**引用格式**:

```markdown
相关理论：[理论名称](文档路径) - [关系说明]
```

**示例**:

- `相关理论：[图的流理论](05-高级理论/02-图的流理论/) - 流理论是连通性理论的扩展`
- `相关理论：[Petri网理论](../10-Petri网理论/) - 可用于建模分布式系统`

### 1.3 学习路径引用

**定义**: 前置知识、后续学习和相关主题的引用

**引用格式**:

```markdown
前置知识：[文档名称](文档路径)
后续学习：[文档名称](文档路径)
相关主题：[主题名称](文档路径)
```

### 4.1 文档内引用位置

**位置1: 概述部分**

```markdown
## 📚 概述 / Overview

**相关文档**：
- 📖 [相关文档1](路径) - [说明]
- 📖 [相关文档2](路径) - [说明]
```

**位置2: 概念定义处**

```markdown
### 1.1 概念定义

**定义**: [定义内容]

参见：[相关概念](路径#锚点) - [关系说明]
```

**位置3: 章节末尾**

```markdown
## 📚 相关链接 / Related Links

### 相关概念
- [概念1](路径) - [说明]
- [概念2](路径) - [说明]

### 相关理论
- [理论1](路径) - [关系说明]
- [理论2](路径) - [关系说明]
```

### 4.2 跨文档引用位置

**位置1: 文档开头**

```markdown
**相关文档**：
- 📖 [前置知识](路径) - 需要先学习

```

