## 📋 **一、任务执行流程 / Task Execution Process**

### 1.1 标准执行流程

```text
1. 查看任务清单 → 2. 检查任务优先级 → 3. 评估资源需求
→ 4. 制定执行计划 → 5. 开始执行任务 → 6. 质量检查
→ 7. 更新进度 → 8. 完成任务
```

### 1.2 快速决策流程

```mermaid
flowchart TD
    A[接到任务] --> B{优先级?}
    B -->|P0| C[立即执行]
    B -->|P1| D{资源充足?}
    B -->|P2| E{时间允许?}
    B -->|P3| F[延后执行]

    D -->|是| C
    D -->|否| G[等待资源]

    E -->|是| C
    E -->|否| F

    C --> H[执行任务]
    H --> I[质量检查]
    I --> J[更新进度]
    J --> K[完成任务]
```
