## 📋 **核心概念速查 / Core Concepts Quick Reference**

### 社会网络基础概念

| 概念 | 定义 | 相关文档 |
|------|------|---------|
| **社会网络** | SN = (A, R, W)，行动者集A、关系集R、权重W | `01-社会网络基础.md` |
| **行动者(Actor)** | 网络中的节点，代表个人、组织等 | `01-社会网络基础.md` |
| **关系(Relation)** | 网络中的边，代表社会关系 | `01-社会网络基础.md` |
| **中心性(Centrality)** | 节点在网络中的重要性度量 | `01-社会网络基础.md` |
| **社区(Community)** | 内部连接密集的节点子集 | `02-网络分析.md` |
| **小世界网络** | 高聚类系数、短平均路径长度的网络 | `01-社会网络基础.md` |
| **无标度网络** | 度分布遵循幂律的网络 | `01-社会网络基础.md` |

### 网络分析概念

| 概念 | 定义 | 相关文档 |
|------|------|---------|
| **度中心性** | 节点的连接数 | `02-网络分析.md` |
| **介数中心性** | 节点在最短路径中的出现频率 | `02-网络分析.md` |
| **接近中心性** | 节点到其他节点的平均距离 | `02-网络分析.md` |
| **特征向量中心性** | 基于邻居重要性的中心性 | `02-网络分析.md` |
| **模块度** | 社区检测质量度量 | `02-网络分析.md` |
| **聚类系数** | 节点邻居间的连接密度 | `02-网络分析.md` |
