## 📊 **二、概念多维矩阵 / Multi-dimensional Concept Matrices**

### 2.1 模型检测算法对比矩阵

| 维度 | 显式状态检测 | 符号模型检测 | 有界模型检测 |
|------|------------|------------|------------|
| **定义** | 显式枚举所有状态 | 使用BDD符号表示状态 | 限制深度使用SAT求解 |
| **关系** | 基础方法 | 显式方法的符号化 | 显式方法的有界版本 |
| **适用场景** | 小规模系统 | 中等规模系统 | 中等规模系统，深度有限 |
| **优缺点** | 简单但状态爆炸 | 压缩状态但需要BDD优化 | 高效但只能检测有限深度 |

### 2.2 时序逻辑对比矩阵

| 维度 | LTL | CTL | CTL* |
|------|-----|-----|------|
| **定义** | 线性时序逻辑 | 计算树逻辑 | 组合逻辑 |
| **关系** | 路径公式 | 状态公式 | 组合LTL和CTL |
| **表达能力** | 路径性质 | 分支性质 | 最强 |
| **检测复杂度** | PSPACE完全 | 线性时间 | 双重指数时间 |

## 🌳 **三、决策树图 / Decision Trees**

### 3.1 模型检测方法选择决策树

```mermaid
flowchart TD
    Start[需要选择模型检测方法] --> Q1{系统规模?}

    Q1 -->|小规模 <10^6状态| Explicit[使用显式状态检测<br/>简单直接]
    Q1 -->|中等规模| Q2{需要检测性质类型?}

    Q2 -->|路径性质| LTL[使用LTL模型检测<br/>线性时序逻辑]
    Q2 -->|分支性质| CTL[使用CTL模型检测<br/>计算树逻辑]
    Q2 -->|复杂性质| Q3{状态空间特征?}

    Q3 -->|规则结构| Symbolic[使用符号模型检测<br/>BDD压缩]
    Q3 -->|深度有限| Bounded[使用有界模型检测<br/>SAT求解]

    Explicit --> Use1[应用: 协议验证<br/>小规模系统]
    LTL --> Use2[应用: 死锁检测<br/>活性性质]
    CTL --> Use3[应用: 可达性分析<br/>安全性性质]
    Symbolic --> Use4[应用: 硬件验证<br/>电路验证]
    Bounded --> Use5[应用: 软件验证<br/>程序验证]

    style Start fill:#f0f0f0
```

## 🔄 **五、控制执行数据流图 / Control Flow & Data Flow Diagrams**

### 5.1 模型检测执行流程

```mermaid
flowchart TD
    Start([开始模型检测]) --> Input[输入: 系统模型M<br/>性质公式φ]

    Input --> Parse[解析模型和公式<br/>构建转换系统]

    Parse --> Build[构建状态空间<br/>或符号表示]

    Build --> Algorithm{选择检测算法}

    Algorithm -->|显式| Explicit[显式状态枚举<br/>DFS/BFS搜索]
    Algorithm -->|符号| Symbolic[符号模型检测<br/>BDD操作]
    Algorithm -->|有界| Bounded[有界模型检测<br/>SAT求解]

    Explicit --> Check[检查性质满足性]
    Symbolic --> Check
    Bounded --> Check

    Check --> Result{结果?}

    Result -->|满足| Satisfy[输出: 性质满足<br/>生成证据路径]
    Result -->|不满足| Counter[输出: 性质不满足<br/>生成反例路径]

    Satisfy --> End([检测结束])
    Counter --> End

    style Start fill:#e1f5ff
    style End fill:#d4edda
    style Counter fill:#fff3cd
```
