## 1.1 Objects and Structures

- **Objects**: Formal proof system $P=\langle L, A, R, T \rangle$, where $L$ is the formal language, $A$ is the axiom set, $R$ is the inference rule set, $T$ is the theorem set
- **Structures**: Proofs, theorems, axioms, inference rules, logical systems, proof strategies

## 1.2 Basic Relations

- $R_{derive}$: Derivation relation
- $R_{equiv}$: Equivalence relation
- $R_{imply}$: Implication relation
- $R_{proof}$: Proof relation

## 1.3 Important Theorems and Reasoning

- **Theorem 1.3.1** (G√∂del's First Incompleteness Theorem): For any recursively enumerable and consistent formal system containing arithmetic, there exists a formula $\phi$ such that neither $\phi$ nor $\neg\phi$ is provable
- **Theorem 1.3.2** (Soundness Theorem): If $\Gamma \vdash_P \phi$, then $\Gamma \models \phi$
- **Theorem 1.3.3** (Completeness Theorem): If $\Gamma \models \phi$, then $\Gamma \vdash_{P_{FOL}} \phi$

## 1.4 Reasoning and Proof System

- Natural deduction reasoning, resolution reasoning, model checking reasoning, type theory reasoning
- Multi-modal representation: logical formulas, proof trees, type terms, Kripke structures

## 1.5 Mapping to Meta-Model

- Formal proof system $P$ is an instance of Meta-model object $P$
- Derivation, equivalence, implication, proof relations are concrete realizations of $R_{derive},R_{equiv},R_{imply},R_{proof}$
