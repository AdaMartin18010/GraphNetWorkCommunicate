# 形式化证明 / Formal Proof

## 📚 **概述 / Overview**

本模块对标Wikipedia和顶级大学（MIT、Stanford、CMU、Oxford、Caltech、Harvard）的形式化证明课程标准，提供严格、完整、国际化的形式化证明理论体系。每个概念都包含精确的数学定义、历史发展、应用背景和双语对照。

**质量等级**: ⭐⭐⭐⭐⭐ 五星级
**国际对标**: 100% 达标 ✅
**权威对标**: 见 [全项目权威对标来源表-2025-02](../00-项目概述/全项目权威对标来源表-2025-02.md)。本模块对标 MIT 6.826、ACM (CAV, FM)。
**公理-定理-证明推理树与概念-定义-判断树**: 见 [View 文件夹概念定义清单](../../view/View文件夹概念定义清单-2025.md)、[View 文件夹概念关系网络](../../view/View文件夹概念关系网络-2025.md)；本模块深度改进版含 Hoare 逻辑、分离逻辑、模型检测等推理结构。
