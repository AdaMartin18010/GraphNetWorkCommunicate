## 🎯 **1. 同伦类型论的多种等价定义 / Multiple Equivalent Definitions**

### 1.1 类型定义

**定义 1.1** (同伦类型论 - 类型定义)

**同伦类型论**（Homotopy Type Theory, HoTT）是将**类型视为拓扑空间**的类型论，由Voevodsky等人在2000年代提出。在HoTT中，类型是空间，项是点，相等类型是路径空间。

**核心思想**：

- **类型即空间**（Types as Spaces）
- **相等即路径**（Equality as Paths）
- **同伦等价**（Homotopy Equivalence）

### 1.2 拓扑定义

**定义 1.2** (同伦类型论 - 拓扑定义)

**同伦类型论**是**类型论**和**代数拓扑学**的结合，使用拓扑学的方法研究类型论，同时用类型论的方法研究拓扑学。

**Univalence公理**：

$$\text{univalence}: (A \simeq B) \simeq (A = B)$$

即，同伦等价和类型相等是等价的。

### 1.3 在形式化数学中的应用

**应用**：

- **数学库形式化**：使用HoTT形式化数学
- **同伦论形式化**：形式化代数拓扑
- **证明助手**：Coq、Agda等支持HoTT

## 2. 核心概念 / Core Concepts

### 2.1 相等类型

在HoTT中，**相等类型** $a =_A b$ 是一个类型，表示 $a$ 和 $b$ 之间的**路径**。

**路径空间**：相等类型的项是连接两点的路径。

### 2.2 同伦等价

**定义 2.1** (同伦等价)

两个类型 $A$ 和 $B$ 是**同伦等价的**，如果存在函数 $f: A \to B$ 和 $g: B \to A$，使得 $g \circ f \simeq \text{id}_A$ 且 $f \circ g \simeq \text{id}_B$。
