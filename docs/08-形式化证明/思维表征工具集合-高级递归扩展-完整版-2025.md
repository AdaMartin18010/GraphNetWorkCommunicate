## 🌲 **四、证明树图 / Proof Trees**

### 4.1 递归定义存在性证明树

```mermaid
graph TD
    Theorem[递归定义存在性:<br/>递归函数f存在<br/>且唯一满足<br/>递归方程] --> FixedPoint[不动点理论:<br/>递归方程对应<br/>函数空间上的<br/>函数算子]

    FixedPoint --> Contract[收缩映射:<br/>证明算子<br/>是收缩映射<br/>Lipschitz常数<1]

    Contract --> Banach[Banach不动点定理:<br/>收缩映射在<br/>完备度量空间<br/>有唯一不动点]

    Banach --> Existence[存在性:<br/>递归函数f存在<br/>是算子的不动点<br/>满足递归方程]

    Existence --> Uniqueness[唯一性:<br/>不动点唯一<br/>递归函数唯一]

    Uniqueness --> Conclusion[结论: 递归定义<br/>存在唯一解<br/>函数f存在且唯一]

    style Theorem fill:#e1f5ff
    style Conclusion fill:#d4edda
    style Contract fill:#fff3cd
```

```mermaid
graph TD

    Supremum --> FixedPoint[不动点:<br/>上确界是F的<br/>最小不动点<br/>满足F(x) = x]

    FixedPoint --> Conclusion[结论: 连续函数F<br/>在完全偏序集<br/>有最小不动点]
```
