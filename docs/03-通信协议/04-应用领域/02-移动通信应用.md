## 📐 **形式化定义 / Formal Definition**

### 定义 4.2.1 (移动通信网络 / Mobile Communication Network)

**移动通信网络**是一个动态的网络系统：

$$\mathcal{M} = \langle B, U, C, H, P \rangle$$

其中：

- $B = \{b_1, b_2, \ldots, b_m\}$ 是基站集合
- $U = \{u_1, u_2, \ldots, u_n\}$ 是用户设备集合
- $C: U \to B$ 是连接函数
- $H: U \times \mathbb{R} \to \mathbb{R}^2$ 是用户移动轨迹
- $P$ 是协议栈

### 定义 4.2.2 (切换过程 / Handover Process)

**切换过程**是用户从一个基站转移到另一个基站的过程：

$$\text{Handover}(u, b_i, b_j) = \{t_{start}, t_{prepare}, t_{execute}, t_{complete}\}$$

## 💻 **算法实现 / Algorithm Implementation**

### 4.2.1 切换决策算法 / Handover Decision Algorithm

```python
class HandoverController:
    """5G切换控制算法"""

    def __init__(self, threshold_rssi=-100, threshold_sinr=10):
        self.threshold_rssi = threshold_rssi  # RSSI阈值
        self.threshold_sinr = threshold_sinr  # SINR阈值
        self.hysteresis = 3  # 迟滞参数

    def should_handover(self, current_cell, candidate_cells):
        """判断是否需要切换"""
        current_rssi = current_cell['rssi']
        current_sinr = current_cell['sinr']

        # 当前信号质量评估
        if current_rssi > self.threshold_rssi and current_sinr > self.threshold_sinr:
            return False, None

        # 查找最佳候选小区
        best_cell = None
        best_score = -float('inf')

        for cell in candidate_cells:
            # 信号强度检查
            if cell['rssi'] < self.threshold_rssi:
                continue

            # 计算切换分数（考虑信号强度和负载）
            score = self._calculate_score(cell)

            # 迟滞处理：避免乒乓切换
            if score > best_score + self.hysteresis:
                best_score = score
                best_cell = cell

        if best_cell and best_score > current_rssi + self.hysteresis:
            return True, best_cell

        return False, None

    def _calculate_score(self, cell):
        """计算小区切换分数"""
        rssi_weight = 0.6
        load_weight = 0.4

        # 归一化RSSI (假设范围 -120 到 -50)
        normalized_rssi = (cell['rssi'] + 120) / 70

        # 归一化负载 (0-1, 1表示无负载)
        normalized_load = 1 - cell['load']

        score = rssi_weight * normalized_rssi + load_weight * normalized_load
        return score
```
