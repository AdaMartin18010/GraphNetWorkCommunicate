## 🎯 **1. 哈密顿性的多种等价定义 / Multiple Equivalent Definitions**

### 1.1 路径定义

**定义 1.1.1** (哈密顿性 - 路径定义)

图的哈密顿路径是经过每个顶点恰好一次的路径。

### 1.2 圈定义

**定义 1.1.2** (哈密顿性 - 圈定义)

图的哈密顿圈是经过每个顶点恰好一次的圈。

### 1.3 闭包定义

**定义 1.1.3** (哈密顿性 - 闭包定义)

图的哈密顿闭包是完全图，当且仅当原图有哈密顿圈。

## 🎯 **2. 哈密顿性的判定定理 / Hamiltonicity Theorems**

### 2.1 Dirac定理

**定理 2.1.1** (Dirac定理)

如果n阶简单图（n≥3）中每个顶点的度至少为n/2，则图是哈密顿图。

### 2.2 Ore定理

**定理 2.2.1** (Ore定理)

如果n阶简单图（n≥3）中对于任意两个不相邻的顶点u和v，有deg(u) + deg(v) ≥ n，则图是哈密顿图。

### 2.3 Chvátal定理

**定理 2.3.1** (Chvátal定理)

如果图的度序列满足Chvátal条件，则图是哈密顿图。
