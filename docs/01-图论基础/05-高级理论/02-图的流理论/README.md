# 图的流理论 / Graph Flow Theory

## 📚 **概述 / Overview**

本目录包含图论中的流理论，包括网络流的基本概念、最大流最小割定理、最大流算法和多源多汇流问题。

*注：基础流理论已在 `02-连通性.md` 第7节中介绍。本节将深入讨论高级内容和算法优化。*
