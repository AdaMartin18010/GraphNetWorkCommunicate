## 📑 **目录结构 / Directory Structure**

本目录按主题组织为以下文件夹结构：

1. **[01-图的分解理论/](01-图的分解理论/)** - 树分解、路径分解、分支分解
   - `01-树分解.md` - 树分解的定义、树宽、算法实现
   - `02-路径分解.md` - 路径分解的定义、路径宽、算法实现
   - `03-分支分解.md` - 分支分解的定义、分支宽、性质
   - `04-分解宽度与算法设计.md` - 分解宽度与算法设计的关系、动态规划应用

2. **[02-图的流理论/](02-图的流理论/)** - 网络流的基本概念、最大流算法
   - `01-网络流的基本概念.md` - 流网络、流、流值、残差网络的定义
   - `02-最大流最小割定理的严格证明.md` - 最大流最小割定理的完整证明
   - `03-最大流算法详细分析.md` - Edmonds-Karp算法和Dinic算法的详细实现
   - `04-多源多汇流问题.md` - 多源多汇流问题的定义和转换方法

3. **[03-图的匹配理论/](03-图的匹配理论/)** - König定理、霍尔婚姻定理、匹配算法
   - `01-König定理的完整证明.md` - König定理的完整证明
   - `02-霍尔婚姻定理的详细证明.md` - 霍尔婚姻定理的详细证明
   - `03-最大匹配算法.md` - 匈牙利算法和Hopcroft-Karp算法
   - `04-加权匹配算法.md` - Kuhn-Munkres算法（KM算法）

4. **[04-极值图论/](04-极值图论/)** - Turán定理、拉姆齐理论、极值结构
   - `01-Turán定理的详细证明.md` - Turán定理的完整证明
   - `02-拉姆齐理论简介.md` - 拉姆齐理论的基本概念和定理
   - `03-图的不等式和极值结构.md` - 基本不等式和极值图的结构
