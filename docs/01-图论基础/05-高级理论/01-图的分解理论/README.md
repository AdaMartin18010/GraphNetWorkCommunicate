# 图的分解理论 / Graph Decomposition Theory

## 📚 **概述 / Overview**

本目录包含图论中的分解理论，包括树分解、路径分解和分支分解。这些理论为算法设计和参数化复杂性提供了重要的理论基础。

4. **[04-分解宽度与算法设计.md](04-分解宽度与算法设计.md)** - 分解宽度与算法设计的关系、动态规划应用
