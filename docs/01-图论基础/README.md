### 🔬 **核心概念**

- **图的基本定义**: 图、有向图、无向图、加权图
- **特殊图类型**: 完全图、二分图、树、平面图
- **图的基本参数**: 阶、大小、度数、密度
- **图的基本关系**: 邻接、路径、连通性
- **图变换操作**: 子图、并、交、补
- **图表示方法**: 邻接矩阵、邻接表、边列表

### 🌐 **重要定理**

- **握手引理**: 度数与边数的关系
- **欧拉定理**: 欧拉回路的充要条件
- **哈密顿定理**: 哈密顿回路的充分条件
- **图分解定理**: 连通分量分解
- **四色定理**: 平面图着色
- **König定理**: 二分图匹配

### 💻 **算法实现**

- **图遍历**: DFS、BFS算法
- **连通性**: 连通分量、强连通分量
- **最短路径**: Dijkstra、Floyd-Warshall
- **最小生成树**: Prim、Kruskal算法
- **图着色**: 贪心着色、回溯着色
- **匹配算法**: 匈牙利算法
