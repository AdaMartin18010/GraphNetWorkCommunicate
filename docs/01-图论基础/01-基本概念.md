## 🎯 **1. 图的基本定义 / Basic Graph Definitions**

### 1.1 图 / Graph

**定义 1.1** (图 / Graph)
**图**是一个有序对 $G = (V, E)$，其中：

- $V$ 是**顶点集**（vertex set），$V \neq \emptyset$
- $E$ 是**边集**（edge set），$E \subseteq V \times V$

**形式化语义**：
$$G \in \mathcal{G} = \{(V, E) \mid V \neq \emptyset \land E \subseteq V \times V\}$$

**历史背景**：

- **1736年**：欧拉解决柯尼斯堡七桥问题，开创图论
- **1852年**：凯莱引入树的概念
- **1936年**：柯尼希出版第一本图论专著

**应用领域**：

- **计算机科学**：网络拓扑、算法设计
- **生物学**：蛋白质相互作用网络
- **社会学**：社交网络分析
- **物理学**：复杂系统建模

### 1.2 有向图 / Directed Graph

**定义 1.2** (有向图 / Directed Graph)
**有向图**是图 $G = (V, E)$，其中边集 $E$ 中的元素是有序对 $(u, v)$，表示从顶点 $u$ 到顶点 $v$ 的有向边。

**形式化定义**：
$$G_{dir} = (V, E) \text{ where } E \subseteq V \times V$$

**性质**：

- 边有方向性
- 邻接矩阵通常不对称
- 支持路径和回路的概念

**应用案例**：

- **依赖关系图**：软件模块依赖
- **状态转换图**：自动机理论
- **食物链网络**：生态学建模

### 1.3 无向图 / Undirected Graph

**定义 1.3** (无向图 / Undirected Graph)
**无向图**是图 $G = (V, E)$，其中边集 $E$ 中的元素是无序对 $\{u, v\}$，表示顶点 $u$ 和 $v$ 之间的无向边。

**形式化定义**：
$$G_{undir} = (V, E) \text{ where } E \subseteq \{\{u, v\} \mid u, v \in V, u \neq v\}$$

**性质**：

- 边无方向性
- 邻接矩阵对称
- 度数概念自然

**应用案例**：

- **社交网络**：朋友关系
- **分子结构**：化学键连接
- **交通网络**：道路连接

## 🔗 **2. 图的特殊类型 / Special Graph Types**

### 2.1 完全图 / Complete Graph

**定义 2.1** (完全图 / Complete Graph)
**完全图** $K_n$ 是包含 $n$ 个顶点的图，其中任意两个不同顶点之间都有一条边。

**形式化定义**：
$$K_n = (V, E) \text{ where } |V| = n \land E = \{\{u, v\} \mid u, v \in V, u \neq v\}$$

**性质**：

- 边数：$|E| = \frac{n(n-1)}{2}$
- 每个顶点度数：$d(v) = n-1$
- 对称性：高度对称

**应用**：

- **网络设计**：最大连通性
- **算法分析**：最坏情况分析
- **组合数学**：极值问题

### 2.2 二分图 / Bipartite Graph

**定义 2.2** (二分图 / Bipartite Graph)
**二分图**是图 $G = (V, E)$，其中顶点集 $V$ 可以划分为两个不相交的子集 $V_1$ 和 $V_2$，使得每条边都连接 $V_1$ 中的一个顶点和 $V_2$ 中的一个顶点。

**形式化定义**：
$$G = (V_1 \cup V_2, E) \text{ where } V_1 \cap V_2 = \emptyset \land E \subseteq V_1 \times V_2$$

**判定定理**：
图 $G$ 是二分图当且仅当 $G$ 不包含奇数长度的环。

**应用**：

- **匹配问题**：婚姻匹配、任务分配
- **网络流**：最大流最小割
- **调度问题**：资源分配

### 2.3 树 / Tree

**定义 2.3** (树 / Tree)
**树**是连通无环的无向图。

**等价定义**：

1. 连通无环图
2. 任意两点间有唯一路径
3. 连通且 $|E| = |V| - 1$
4. 无环且 $|E| = |V| - 1$

**性质**：

- 顶点数：$n$
- 边数：$n-1$
- 连通分量数：1
- 环数：0

**应用**：

- **数据结构**：二叉树、B树
- **网络设计**：最小生成树
- **算法设计**：动态规划

## 📊 **3. 图的基本参数 / Basic Graph Parameters**

### 3.1 图的阶和大小 / Order and Size

**定义 3.1** (图的阶 / Order of Graph)
图 $G = (V, E)$ 的**阶**是顶点集 $V$ 的基数：
$$|G| = |V|$$

**定义 3.2** (图的大小 / Size of Graph)
图 $G = (V, E)$ 的**大小**是边集 $E$ 的基数：
$$\|G\| = |E|$$

**关系**：

- 对于树：$\|G\| = |G| - 1$
- 对于完全图：$\|G\| = \frac{|G|(|G|-1)}{2}$

### 3.2 顶点的度 / Degree of Vertex

**定义 3.3** (顶点的度 / Degree of Vertex)
在无向图中，顶点 $v$ 的**度**是与 $v$ 关联的边数：
$$d(v) = |\{e \in E \mid v \in e\}|$$

**定义 3.4** (入度和出度 / In-degree and Out-degree)
在有向图中，顶点 $v$ 的**入度**和**出度**分别为：
$$d^-(v) = |\{(u, v) \in E\}|, \quad d^+(v) = |\{(v, u) \in E\}|$$

**性质**：

- 总度数：$\sum_{v \in V} d(v) = 2|E|$（无向图）
- 总入度 = 总出度 = $|E|$（有向图）

### 3.3 握手引理 / Handshaking Lemma

**定理 3.1** (握手引理 / Handshaking Lemma)
在无向图中，所有顶点的度数之和等于边数的两倍：
$$\sum_{v \in V} d(v) = 2|E|$$

**证明**：

1. 每条边贡献2度（一个给每个端点）
2. 因此总度数 = 2 × 边数
3. 由集合论：$\sum_{v \in V} d(v) = \sum_{v \in V} |\{e \in E \mid v \in e\}| = 2|E|$

**推论**：

- 图中奇数度顶点个数为偶数
- 正则图中所有顶点度数相同

## 🔍 **4. 图的基本关系 / Basic Graph Relations**

### 4.1 邻接关系 / Adjacency Relation

**定义 4.1** (邻接关系 / Adjacency Relation)
在无向图中，顶点 $u$ 和 $v$ 是**邻接的**当且仅当 $\{u, v\} \in E$：
$$R_{adj}(u, v) \iff \{u, v\} \in E$$

在有向图中，顶点 $u$ 邻接到顶点 $v$ 当且仅当 $(u, v) \in E$：
$$R_{adj}(u, v) \iff (u, v) \in E$$

**性质**：

1. **自反性**：在无向图中，$R_{adj}$ 是自反的当且仅当图包含自环
2. **对称性**：在无向图中，$R_{adj}$ 是对称的
3. **传递性**：$R_{adj}$ 一般不满足传递性

### 4.2 路径关系 / Path Relation

**定义 4.2** (路径 / Path)
**路径**是顶点序列 $P = (v_0, v_1, \ldots, v_k)$，其中：

- 对于 $i = 0, 1, \ldots, k-1$，$(v_i, v_{i+1}) \in E$（有向图）或 $\{v_i, v_{i+1}\} \in E$（无向图）
- 路径长度为 $k$

**形式化定义**：
$$P = (v_0, v_1, \ldots, v_k) \text{ where } \forall i \in [0, k-1]: (v_i, v_{i+1}) \in E$$

**定义 4.3** (路径关系 / Path Relation)
顶点 $u$ 和 $v$ 之间存在**路径关系**当且仅当存在从 $u$ 到 $v$ 的路径：
$$R_{path}(u, v) \iff \exists P: u \stackrel{P}{\to} v$$

**性质**：

- **传递性**：$R_{path}(u, v) \land R_{path}(v, w) \implies R_{path}(u, w)$
- **自反性**：$R_{path}(u, u)$（空路径）
- **对称性**：在无向图中，$R_{path}(u, v) \iff R_{path}(v, u)$

### 4.3 连通性 / Connectivity

**定义 4.4** (连通图 / Connected Graph)
无向图 $G$ 是**连通的**当且仅当任意两个顶点之间都存在路径。

**形式化定义**：
$$\text{connected}(G) \iff \forall u, v \in V: R_{path}(u, v)$$

**定义 4.5** (强连通图 / Strongly Connected Graph)
有向图 $G$ 是**强连通的**当且仅当任意两个顶点之间都存在有向路径。

**形式化定义**：
$$\text{strongly\_connected}(G) \iff \forall u, v \in V: R_{path}(u, v) \land R_{path}(v, u)$$

**连通分量**：

- **连通分量**：极大连通子图
- **强连通分量**：极大强连通子图

## 🔄 **5. 图的变换操作 / Graph Transformation Operations**

### 5.1 子图 / Subgraph

**定义 5.1** (子图 / Subgraph)
图 $H = (V_H, E_H)$ 是图 $G = (V_G, E_G)$ 的**子图**当且仅当：
$$V_H \subseteq V_G \land E_H \subseteq E_G \cap (V_H \times V_H)$$

**形式化定义**：
$$H \subseteq G \iff V_H \subseteq V_G \land E_H \subseteq E_G \cap (V_H \times V_H)$$

**定义 5.2** (诱导子图 / Induced Subgraph)
图 $H = (V_H, E_H)$ 是图 $G = (V_G, E_G)$ 的**诱导子图**当且仅当：
$$V_H \subseteq V_G \land E_H = E_G \cap (V_H \times V_H)$$

**性质**：

1. **自反性**：$G \subseteq G$
2. **传递性**：$H \subseteq G \land K \subseteq H \implies K \subseteq G$
3. **反对称性**：$H \subseteq G \land G \subseteq H \implies H = G$

### 5.2 图的并和交 / Graph Union and Intersection

**定义 5.3** (图的并 / Graph Union)
图 $G_1 = (V_1, E_1)$ 和 $G_2 = (V_2, E_2)$ 的**并**是：
$$G_1 \cup G_2 = (V_1 \cup V_2, E_1 \cup E_2)$$

**定义 5.4** (图的交 / Graph Intersection)
图 $G_1 = (V_1, E_1)$ 和 $G_2 = (V_2, E_2)$ 的**交**是：
$$G_1 \cap G_2 = (V_1 \cap V_2, E_1 \cap E_2)$$

### 5.3 图的补图 / Graph Complement

**定义 5.5** (图的补图 / Graph Complement)
图 $G = (V, E)$ 的**补图**是：
$$\overline{G} = (V, \overline{E}) \text{ where } \overline{E} = \{\{u, v\} \mid u, v \in V, u \neq v, \{u, v\} \notin E\}$$

**性质**：

- $G \cup \overline{G} = K_n$（完全图）
- $G \cap \overline{G} = \emptyset$（空图）
- $\overline{\overline{G}} = G$

## 🧮 **6. 图的表示方法 / Graph Representation Methods**

### 6.1 邻接矩阵 / Adjacency Matrix

**定义 6.1** (邻接矩阵 / Adjacency Matrix)
图 $G = (V, E)$ 的邻接矩阵 $A$ 是 $|V| \times |V|$ 矩阵：
$$A_{ij} = \begin{cases}
1 & \text{if } (v_i, v_j) \in E \\
0 & \text{otherwise}
\end{cases}$$

**性质**：
1. 对于无向图，$A$ 是对称的
2. 对于无环图，$A$ 的对角线元素为0
3. $A^k$ 的第 $(i,j)$ 元素表示从 $v_i$ 到 $v_j$ 的长度为 $k$ 的路径数

**复杂度**：
- 空间复杂度：$O(|V|^2)$
- 查询边存在性：$O(1)$
- 遍历邻接顶点：$O(|V|)$

### 6.2 邻接表 / Adjacency List

**定义 6.2** (邻接表 / Adjacency List)
图 $G = (V, E)$ 的邻接表是数组 $Adj$，其中 $Adj[v]$ 是与顶点 $v$ 邻接的顶点列表。

**实现**：
```python
class Graph:
    def __init__(self, vertices):
        self.V = vertices
        self.adj = [[] for _ in range(vertices)]

    def add_edge(self, u, v):
        self.adj[u].append(v)
        self.adj[v].append(u)  # 无向图
```

**复杂度**：
- 空间复杂度：$O(|V| + |E|)$
- 查询边存在性：$O(\deg(v))$
- 遍历邻接顶点：$O(\deg(v))$

### 6.3 边列表 / Edge List

**定义 6.3** (边列表 / Edge List)
图 $G = (V, E)$ 的边列表是边集 $E$ 的列表表示。

**实现**：
```python
class Graph:
    def __init__(self):
        self.edges = []

    def add_edge(self, u, v):
        self.edges.append((u, v))
```

**复杂度**：
- 空间复杂度：$O(|E|)$
- 查询边存在性：$O(|E|)$
- 遍历所有边：$O(|E|)$

## 🎯 **7. 重要定理和性质 / Important Theorems and Properties**

### 7.1 欧拉定理 / Euler's Theorem

**定理 7.1** (欧拉定理 / Euler's Theorem)
连通无向图存在欧拉回路的充要条件是所有顶点的度都是偶数。

**形式化表述**：
$$\text{connected}(G) \land (\forall v \in V: d(v) \text{ is even}) \iff \text{has\_euler\_circuit}(G)$$

**历史背景**：
- **1736年**：欧拉解决柯尼斯堡七桥问题
- **1873年**：Hierholzer给出构造性证明
- **现代应用**：DNA测序、电路设计

**证明**：
1. **必要性**：如果存在欧拉回路，则每个顶点被访问偶数次
2. **充分性**：通过构造法证明，详见欧拉定理严格证明文档

### 7.2 哈密顿定理 / Hamiltonian Theorem

**定理 7.2** (哈密顿定理 / Hamiltonian Theorem)
完全图 $K_n$（$n \geq 3$）存在哈密顿回路。

**形式化表述**：
$$n \geq 3 \implies \text{has\_hamiltonian\_cycle}(K_n)$$

**应用**：
- **旅行商问题**：NP完全问题
- **电路设计**：最小成本回路
- **网络优化**：最优路径规划

### 7.3 图的分解定理 / Graph Decomposition Theorem

**定理 7.3** (图的分解定理 / Graph Decomposition Theorem)
任意图都可以唯一分解为若干连通分量。

**形式化表述**：
$$\forall G: \exists! \{C_1, C_2, \ldots, C_k\}: G = \bigcup_{i=1}^k C_i \land \text{connected}(C_i) \land C_i \cap C_j = \emptyset \text{ for } i \neq j$$

**证明**：
1. 定义等价关系：$u \sim v \iff R_{path}(u, v) \lor R_{path}(v, u)$
2. 连通分量是等价类
3. 图的顶点集是等价类的并集

## 🔧 **8. 算法实现 / Algorithm Implementation**

### 8.1 图的遍历算法 / Graph Traversal Algorithms

**算法 8.1** (深度优先搜索 / Depth-First Search)
```python
def dfs(graph, start, visited=None):
    if visited is None:
        visited = set()

    visited.add(start)
    print(start, end=' ')

    for neighbor in graph[start]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)

    return visited
```

**算法 8.2** (广度优先搜索 / Breadth-First Search)
```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    visited.add(start)

    while queue:
        vertex = queue.popleft()
        print(vertex, end=' ')

        for neighbor in graph[vertex]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

    return visited
```

### 8.2 连通性检测算法 / Connectivity Detection Algorithm

**算法 8.3** (连通分量检测 / Connected Components Detection)
```python
def find_connected_components(graph):
    visited = set()
    components = []

    for vertex in graph:
        if vertex not in visited:
            component = set()
            dfs_component(graph, vertex, visited, component)
            components.append(component)

    return components

def dfs_component(graph, vertex, visited, component):
    visited.add(vertex)
    component.add(vertex)

    for neighbor in graph[vertex]:
        if neighbor not in visited:
            dfs_component(graph, neighbor, visited, component)
```
