# 分布式系统 / Distributed Systems

## 📚 **概述 / Overview**

本模块对标Wikipedia和顶级大学（MIT、Stanford、CMU、Oxford、Caltech、Harvard）的分布式系统课程标准，提供严格、完整、国际化的分布式系统理论体系。每个概念都包含精确的数学定义、历史发展、应用背景和双语对照。

**权威对标**: 见 [全项目权威对标来源表-2025-02](../00-项目概述/全项目权威对标来源表-2025-02.md)。本模块对标 MIT 6.824、教材 Tanenbaum *Distributed Systems*、ACM 会议（OSDI、SIGOPS 等）。
**公理-定理-证明推理树与概念-定义-判断树**: 见 [View 文件夹概念定义清单](../../view/View文件夹概念定义清单-2025.md)、[View 文件夹概念关系网络](../../view/View文件夹概念关系网络-2025.md)；本模块 02-一致性协议 含 CAP、Paxos、Raft 等推理与对比矩阵。
**思维表征与 View 双向绑定**: 见 [View 文件夹思维表征工具集-2025](../../view/View文件夹思维表征工具集-2025.md)、[View 文件夹对比矩阵集-2025](../../view/View文件夹对比矩阵集-2025.md)。
**理论关系与跨模块索引**：见 [12-理论关系与认知框架](../12-理论关系与认知框架/README.md)。
**核心定理证明或引用**：CAP 定理、Paxos/Raft 正确性证明或权威引用见 [模块权威对标与缺口清单-2025-02](../00-项目概述/模块权威对标与缺口清单-2025-02.md) §04 缺口细化（待补项：建议补至 01-分布式基础、99-理论应用与案例）。
