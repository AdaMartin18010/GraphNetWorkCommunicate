## 📐 **形式化定义 / Formal Definition**

### 定义 4.4.1 (金融网络 / Financial Network)

**金融网络**是用于建模金融实体及其关系的图结构，可以形式化为：

$$\mathcal{F} = \langle V_{fin}, E_{fin}, \mathcal{W}, \mathcal{T} \rangle$$

其中：

- $V_{fin} = \{v_1, v_2, \ldots, v_n\}$ 是金融实体集合（银行、公司、投资者等）
- $E_{fin} \subseteq V_{fin} \times V_{fin}$ 是金融关系集合（借贷、投资、交易等）
- $\mathcal{W}: E_{fin} \to \mathbb{R}^+$ 是边权重函数，表示关系强度或金额
- $\mathcal{T}: E_{fin} \to \{\text{借贷}, \text{投资}, \text{交易}, \ldots\}$ 是边类型函数

### 定义 4.4.2 (风险传播网络 / Risk Propagation Network)

**风险传播网络**是一个带权有向图 $G_{risk} = (V_{fin}, E_{risk}, W_{risk})$，其中：

- $V_{fin}$ 是金融实体集合
- $E_{risk} = \{(v_i, v_j) | \text{实体 } v_i \text{ 的风险可能传播到 } v_j\}$
- $W_{risk}: E_{risk} \to [0, 1]$ 是风险传播概率函数

风险传播的动态过程可以建模为：

$$R_i(t+1) = R_i(t) + \sum_{j \in \mathcal{N}(i)} w_{ji} \cdot R_j(t) \cdot (1 - R_i(t))$$

其中 $R_i(t)$ 是实体 $i$ 在时间 $t$ 的风险水平。

### 定义 4.4.3 (投资组合网络 / Portfolio Network)

**投资组合网络**是一个二部图 $G_{port} = (V_{investor} \cup V_{asset}, E_{hold})$，其中：

- $V_{investor} = \{i_1, i_2, \ldots, i_m\}$ 是投资者集合
- $V_{asset} = \{a_1, a_2, \ldots, a_n\}$ 是资产集合
- $E_{hold} = \{(i, a) | \text{投资者 } i \text{ 持有资产 } a\}$
- 每条边 $(i, a)$ 带有权重 $w_{ia}$，表示持有数量或价值

### 形式化语义 / Formal Semantics

- **图论语义**: 金融网络作为图结构，节点表示金融实体，边表示金融关系
- **概率语义**: 边权重可以表示交易概率、风险传播概率等
- **时序语义**: 网络结构随时间演化，反映金融市场的动态性

## 🔧 **理论基础 / Theoretical Foundation**

### 4.4.1 金融网络拓扑特征 / Financial Network Topological Features

金融网络具有以下特征：

1. **核心-边缘结构**: 少数核心金融机构连接大量边缘机构
   - 定义: 度分布高度偏斜，少数节点度很高
   - 意义: 系统性风险集中在核心机构

2. **社区结构**: 金融机构按地域、业务类型形成社区
   - 定义: 模块度 $Q > 0.3$ 表示明显的社区结构
   - 意义: 风险在社区内传播更快

3. **小世界特性**: 平均路径长度短，但聚类系数高
   - 定义: $L_{avg} \approx 3-4$，$C \gg C_{random}$
   - 意义: 风险可以快速传播到整个网络

### 4.4.2 风险传播模型 / Risk Propagation Models

#### 4.4.2.1 传染模型 (Contagion Model)

最简单的风险传播模型：

$$R_i(t+1) = \min\left(1, R_i(t) + \beta \sum_{j \in \mathcal{N}(i)} w_{ji} R_j(t)\right)$$

其中 $\beta$ 是传播率。

#### 4.4.2.2 阈值模型 (Threshold Model)

实体在风险超过阈值时才会传播风险：

$$
R_i(t+1) = \begin{cases}
1 & \text{if } \sum_{j \in \mathcal{N}(i)} w_{ji} R_j(t) > \theta_i \\
R_i(t) & \text{otherwise}
\end{cases}
$$

其中 $\theta_i$ 是实体 $i$ 的风险阈值。

### 4.4.3 投资组合优化理论 / Portfolio Optimization Theory

#### 4.4.3.1 现代投资组合理论 (Modern Portfolio Theory)

Markowitz均值-方差优化：

$$\min_{\mathbf{w}} \mathbf{w}^T \Sigma \mathbf{w} - \lambda \mathbf{w}^T \boldsymbol{\mu}$$

其中：

- $\mathbf{w}$ 是资产权重向量
- $\Sigma$ 是资产协方差矩阵
- $\boldsymbol{\mu}$ 是资产期望收益向量
- $\lambda$ 是风险厌恶系数

#### 4.4.3.2 基于网络的优化

使用图神经网络学习资产相关性：

$$\Sigma_{ij} = f_{\text{GNN}}(\mathbf{x}_i, \mathbf{x}_j, G)$$

其中 $G$ 是资产关系网络，$\mathbf{x}_i$ 是资产 $i$ 的特征。

### 算法 4.4.1 (风险传播预测算法 / Risk Propagation Prediction Algorithm)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GATConv, GCNConv
from typing import Tuple, List
import numpy as np

class RiskPropagationPredictor(nn.Module):
    """风险传播预测器 - 使用自适应图神经网络"""

    def __init__(self, node_dim: int = 64, hidden_dim: int = 32,
                 num_layers: int = 3, num_heads: int = 8, dropout: float = 0.2):
        super(RiskPropagationPredictor, self).__init__()

        # 图注意力层
        self.conv1 = GATConv(node_dim, hidden_dim, heads=num_heads, dropout=dropout)
        self.conv2 = GATConv(hidden_dim * num_heads, hidden_dim, heads=1, dropout=dropout)

        # 自适应风险传播层
        self.risk_propagation = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(hidden_dim, hidden_dim)
        )

        # 风险预测器
        self.risk_predictor = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1),
            nn.Sigmoid()
        )

        self.dropout = nn.Dropout(dropout)

    def forward(self, x, edge_index, edge_attr=None, initial_risk=None):
        """
        前向传播

        Args:
            x: 节点特征矩阵 [N, node_dim]
            edge_index: 边索引 [2, E]
            edge_attr: 边属性（风险传播权重）[E, 1] (可选)
            initial_risk: 初始风险水平 [N, 1] (可选)

        Returns:
            risk_scores: 风险分数 [N, 1]
        """
        # 图卷积
        h = F.elu(self.conv1(x, edge_index))
        h = self.dropout(h)
        h = self.conv2(h, edge_index)

        # 如果有初始风险，进行风险传播
        if initial_risk is not None:
            # 风险传播：结合邻居节点的风险
            risk_propagated = self.risk_propagation(h)
            risk_propagated = risk_propagated * initial_risk
            h = h + risk_propagated

        # 预测风险分数
        risk_scores = self.risk_predictor(h)

        return risk_scores

    def predict_risk_propagation(self, x, edge_index, initial_risk, num_steps: int = 10):
        """
        预测多步风险传播

        Args:
            x: 节点特征矩阵
            edge_index: 边索引
            initial_risk: 初始风险水平 [N, 1]
            num_steps: 传播步数

        Returns:
            risk_history: 风险传播历史 [num_steps, N, 1]
        """
        risk_history = [initial_risk]
        current_risk = initial_risk

        for step in range(num_steps):
            risk_scores = self.forward(x, edge_index, initial_risk=current_risk)
            current_risk = risk_scores
            risk_history.append(current_risk)

        return torch.stack(risk_history)
```

### 算法 4.4.2 (投资组合优化算法 / Portfolio Optimization Algorithm)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GCNConv
from typing import Tuple
import numpy as np

class PortfolioOptimizer(nn.Module):
    """投资组合优化器 - 使用自适应GNN学习资产相关性"""

    def __init__(self, asset_dim: int = 32, hidden_dim: int = 16,
                 num_layers: int = 2):
        super(PortfolioOptimizer, self).__init__()

        # 资产特征编码器
        self.encoder = nn.Sequential(
            nn.Linear(asset_dim, hidden_dim * 2),
            nn.ReLU(),
            nn.Linear(hidden_dim * 2, hidden_dim)
        )

        # 图卷积层：学习资产相关性
        self.convs = nn.ModuleList()
        for _ in range(num_layers):
            self.convs.append(GCNConv(hidden_dim, hidden_dim))

        # 相关性预测器
        self.correlation_predictor = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1),
            nn.Tanh()  # 输出范围 [-1, 1]
        )

        # 收益预测器
        self.return_predictor = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1)
        )

    def forward(self, asset_features, asset_graph):
        """
        前向传播

        Args:
            asset_features: 资产特征矩阵 [N, asset_dim]
            asset_graph: 资产关系图（edge_index）

        Returns:
            correlations: 资产相关性矩阵 [N, N]
            returns: 预测收益 [N, 1]
        """
        # 编码资产特征
        h = self.encoder(asset_features)

        # 图卷积
        for conv in self.convs:
            h = F.relu(conv(h, asset_graph))

        # 预测资产相关性
        N = h.size(0)
        correlations = torch.zeros(N, N)
        for i in range(N):
            for j in range(N):
                if i != j:
                    pair_features = torch.cat([h[i], h[j]], dim=-1)
                    correlations[i, j] = self.correlation_predictor(pair_features).squeeze()
                else:
                    correlations[i, j] = 1.0

        # 预测收益
        returns = self.return_predictor(h)

        return correlations, returns

    def optimize_portfolio(self, asset_features, asset_graph,
                          risk_aversion: float = 0.5,
                          target_return: float = None):
        """
        优化投资组合

        Args:
            asset_features: 资产特征矩阵
            asset_graph: 资产关系图
            risk_aversion: 风险厌恶系数
            target_return: 目标收益（可选）

        Returns:
            weights: 最优权重 [N, 1]
        """
        correlations, returns = self.forward(asset_features, asset_graph)

        # 构建协方差矩阵（简化：从相关性矩阵估计）
        # 实际应用中需要更复杂的协方差估计
        std_devs = torch.ones(returns.size(0)) * 0.2  # 假设标准差
        covariance = correlations * std_devs.unsqueeze(0) * std_devs.unsqueeze(1)

        # Markowitz优化（简化版本）
        # 实际应用中需要使用优化器求解
        N = returns.size(0)

        # 使用梯度下降优化权重
        weights = nn.Parameter(torch.ones(N) / N)
        optimizer = torch.optim.Adam([weights], lr=0.01)

        for _ in range(100):
            # 归一化权重
            w = F.softmax(weights, dim=0)

            # 计算组合收益和风险
            portfolio_return = (w * returns.squeeze()).sum()
            portfolio_risk = torch.sqrt((w.unsqueeze(0) @ covariance @ w.unsqueeze(1)).squeeze())

            # 目标函数：最大化收益，最小化风险
            if target_return is not None:
                loss = -portfolio_return + risk_aversion * portfolio_risk + \
                       10 * F.relu(target_return - portfolio_return)
            else:
                loss = -portfolio_return + risk_aversion * portfolio_risk

            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

        return F.softmax(weights, dim=0)
```

### 算法 4.4.3 (欺诈检测算法 / Fraud Detection Algorithm)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GCNConv, GATConv
from typing import Tuple

class FraudDetector(nn.Module):
    """欺诈检测器 - 使用自适应GNN分析交易网络"""

    def __init__(self, node_dim: int = 32, hidden_dim: int = 16,
                 num_layers: int = 3):
        super(FraudDetector, self).__init__()

        # 图卷积层
        self.convs = nn.ModuleList()
        self.convs.append(GCNConv(node_dim, hidden_dim))
        for _ in range(num_layers - 2):
            self.convs.append(GCNConv(hidden_dim, hidden_dim))
        self.convs.append(GCNConv(hidden_dim, hidden_dim))

        # 异常检测器
        self.anomaly_detector = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1),
            nn.Sigmoid()
        )

        # 特征重构器（用于异常检测）
        self.reconstructor = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim * 2),
            nn.ReLU(),
            nn.Linear(hidden_dim * 2, node_dim)
        )

    def forward(self, x, edge_index):
        """
        前向传播

        Args:
            x: 节点特征矩阵 [N, node_dim]
            edge_index: 边索引 [2, E]

        Returns:
            fraud_scores: 欺诈分数 [N, 1]
            reconstructed_features: 重构特征 [N, node_dim]
        """
        # 图卷积
        h = x
        for conv in self.convs:
            h = F.relu(conv(h, edge_index))

        # 预测欺诈分数
        fraud_scores = self.anomaly_detector(h)

        # 重构特征（用于异常检测）
        reconstructed_features = self.reconstructor(h)

        return fraud_scores, reconstructed_features

    def detect_fraud(self, x, edge_index, threshold: float = 0.5):
        """
        检测欺诈

        Args:
            x: 节点特征矩阵
            edge_index: 边索引
            threshold: 欺诈阈值

        Returns:
            fraud_indices: 欺诈节点索引列表
            fraud_scores: 所有节点的欺诈分数
        """
        fraud_scores, _ = self.forward(x, edge_index)
        fraud_indices = (fraud_scores.squeeze() > threshold).nonzero(as_tuple=True)[0].tolist()

        return fraud_indices, fraud_scores
```

## 📊 **性能评估 / Performance Evaluation**

### 4.4.1 评估指标 / Evaluation Metrics

#### 风险传播预测指标

1. **准确率指标**:
   - **风险预测准确率**: $\text{Acc} = \frac{TP + TN}{TP + TN + FP + FN}$
   - **风险传播路径预测准确率**: 正确预测风险传播路径的比例
   - **系统性风险识别率**: 提前识别系统性风险的比例

2. **时间指标**:
   - **提前预警时间**: 风险发生前的预警时间
   - **响应时间**: 系统响应风险事件的时间

#### 投资组合优化指标

1. **收益指标**:
   - **年化收益率**: $\text{Return} = \frac{V_{end} - V_{start}}{V_{start}} \times \frac{252}{T}$
   - **夏普比率**: $\text{Sharpe} = \frac{E[R_p] - R_f}{\sigma_p}$
   - **信息比率**: $\text{IR} = \frac{E[R_p] - R_b]}{\sigma(R_p - R_b)}$

2. **风险指标**:
   - **波动率**: $\sigma_p = \sqrt{\text{Var}(R_p)}$
   - **最大回撤**: $\text{MDD} = \max_{t} \frac{V_{peak} - V_t}{V_{peak}}$
   - **VaR (Value at Risk)**: 在给定置信水平下的最大损失

#### 欺诈检测指标

1. **分类指标**:
   - **精确率**: $\text{Precision} = \frac{TP}{TP + FP}$
   - **召回率**: $\text{Recall} = \frac{TP}{TP + FN}$
   - **F1分数**: $\text{F1} = \frac{2 \cdot \text{Precision} \cdot \text{Recall}}{\text{Precision} + \text{Recall}}$
   - **AUC-ROC**: 受试者工作特征曲线下面积

2. **业务指标**:
   - **误报率**: $\text{FPR} = \frac{FP}{FP + TN}$
   - **检测延迟**: 从欺诈发生到检测到的时间
   - **成本节约**: 通过早期检测避免的损失

### 4.4.3 复杂度分析 / Complexity Analysis

#### 时间复杂度

1. **风险传播预测**:
   - 图卷积: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d))$
   - 风险传播: $O(T \cdot |E|)$，其中 $T$ 是传播步数
   - **总复杂度**: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d) + T \cdot |E|)$

2. **投资组合优化**:
   - 相关性学习: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d) + |V|^2 \cdot d)$
   - 优化求解: $O(I \cdot |V|^3)$，其中 $I$ 是迭代次数
   - **总复杂度**: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d) + |V|^2 \cdot d + I \cdot |V|^3)$

#### 空间复杂度

- **模型参数**: $O(L \cdot d^2)$
- **中间表示**: $O(|V| \cdot d + |E|)$
- **总空间复杂度**: $O(L \cdot d^2 + |V| \cdot d + |E|)$
