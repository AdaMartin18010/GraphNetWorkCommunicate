## 📐 **形式化定义 / Formal Definition**

### 定义 4.3.1 (生物信息学网络 / Bioinformatics Network)

**生物信息学网络**是用于建模和分析生物系统的图结构，可以形式化为：

$$\mathcal{B} = \langle V, E, \mathcal{F}, \mathcal{A} \rangle$$

其中：

- $V$ 是生物实体节点集（蛋白质、基因、代谢物等）
- $E \subseteq V \times V$ 是生物关系边集（相互作用、调控、共表达等）
- $\mathcal{F}: V \to \mathbb{R}^d$ 是节点特征映射函数
- $\mathcal{A}$ 是自适应机制集，用于根据数据动态调整网络结构

### 定义 4.3.2 (蛋白质相互作用网络 / Protein-Protein Interaction Network)

**蛋白质相互作用网络**是一个无向图 $G_{PPI} = (V_{prot}, E_{int})$，其中：

- $V_{prot} = \{p_1, p_2, \ldots, p_n\}$ 是蛋白质集合
- $E_{int} = \{(p_i, p_j) | \text{蛋白质 } p_i \text{ 与 } p_j \text{ 相互作用}\}$
- 每条边 $(p_i, p_j)$ 可以带有权重 $w_{ij} \in [0, 1]$，表示相互作用的置信度

### 定义 4.3.3 (基因调控网络 / Gene Regulatory Network)

**基因调控网络**是一个有向图 $G_{GRN} = (V_{gene}, E_{reg})$，其中：

- $V_{gene} = \{g_1, g_2, \ldots, g_n\}$ 是基因集合
- $E_{reg} = \{(g_i, g_j) | \text{基因 } g_i \text{ 调控基因 } g_j\}$
- 每条边 $(g_i, g_j)$ 带有调控类型：激活（$+$）或抑制（$-$），以及调控强度 $s_{ij} \in \mathbb{R}$

### 形式化语义 / Formal Semantics

- **图论语义**: 生物网络作为图结构，节点表示生物实体，边表示生物关系
- **概率语义**: 边权重可以表示相互作用的概率或置信度
- **动态语义**: 网络结构可以随时间演化，反映生物过程的动态性

## 🔧 **理论基础 / Theoretical Foundation**

### 4.3.1 网络拓扑特征 / Network Topological Features

生物网络具有以下拓扑特征：

1. **小世界特性**: 平均路径长度短，聚类系数高
   - 定义: $L_{avg} \approx \log N$，$C \gg C_{random}$
   - 意义: 生物系统具有高效的信号传递机制

2. **无标度特性**: 度分布遵循幂律分布
   - 定义: $P(k) \sim k^{-\gamma}$，其中 $\gamma \approx 2-3$
   - 意义: 少数关键节点（hub节点）控制整个网络

3. **模块化结构**: 网络具有明显的社区结构
   - 定义: 模块度 $Q = \frac{1}{2m}\sum_{ij}[A_{ij} - \frac{k_i k_j}{2m}]\delta(c_i, c_j)$
   - 意义: 功能相关的生物实体聚集在一起

### 4.3.2 自适应机制 / Adaptive Mechanisms

在生物信息学网络中，自适应机制包括：

1. **结构自适应**: 根据新发现的相互作用动态更新网络结构
   $$G(t+1) = \text{Adapt}(G(t), \mathcal{D}_{new}(t))$$

2. **特征自适应**: 根据实验数据更新节点特征表示
   $$f_v(t+1) = \text{Update}(f_v(t), x_v^{exp}(t))$$

3. **权重自适应**: 根据置信度更新边权重
   $$w_{ij}(t+1) = \alpha \cdot w_{ij}(t) + (1-\alpha) \cdot c_{ij}^{new}$$

### 算法 4.3.1 (蛋白质相互作用预测算法 / Protein Interaction Prediction Algorithm)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GCNConv, GATConv
from torch_geometric.data import Data
from typing import Tuple, Dict, List
import numpy as np

class ProteinInteractionPredictor(nn.Module):
    """蛋白质相互作用预测器 - 使用自适应图神经网络"""

    def __init__(self, protein_dim: int = 128, hidden_dim: int = 64,
                 num_layers: int = 3, dropout: float = 0.2):
        super(ProteinInteractionPredictor, self).__init__()

        # 图卷积层
        self.convs = nn.ModuleList()
        self.convs.append(GCNConv(protein_dim, hidden_dim))
        for _ in range(num_layers - 2):
            self.convs.append(GCNConv(hidden_dim, hidden_dim))
        self.convs.append(GCNConv(hidden_dim, hidden_dim))

        # 自适应注意力机制
        self.attention = nn.MultiheadAttention(hidden_dim, num_heads=8, dropout=dropout)

        # 预测层
        self.predictor = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1),
            nn.Sigmoid()
        )

        self.dropout = nn.Dropout(dropout)

    def forward(self, x, edge_index, edge_attr=None):
        """
        前向传播

        Args:
            x: 节点特征矩阵 [N, protein_dim]
            edge_index: 边索引 [2, E]
            edge_attr: 边属性 [E, edge_dim] (可选)

        Returns:
            interaction_probs: 相互作用概率 [N*(N-1)//2, 1]
        """
        # 图卷积
        h = x
        for i, conv in enumerate(self.convs):
            h = conv(h, edge_index)
            if i < len(self.convs) - 1:
                h = F.relu(h)
                h = self.dropout(h)

        # 自适应注意力
        h = h.unsqueeze(0)  # [1, N, hidden_dim]
        h_att, _ = self.attention(h, h, h)
        h = h_att.squeeze(0)  # [N, hidden_dim]

        # 预测所有节点对的相互作用
        N = h.size(0)
        interactions = []
        for i in range(N):
            for j in range(i + 1, N):
                pair_features = torch.cat([h[i], h[j]], dim=-1)
                interaction_prob = self.predictor(pair_features)
                interactions.append(interaction_prob)

        return torch.stack(interactions) if interactions else torch.tensor([])

    def predict_interactions(self, x, edge_index, protein_pairs: List[Tuple[int, int]]):
        """
        预测指定蛋白质对的相互作用

        Args:
            x: 节点特征矩阵
            edge_index: 边索引
            protein_pairs: 蛋白质对列表 [(i, j), ...]

        Returns:
            probs: 相互作用概率列表
        """
        h = x
        for conv in self.convs:
            h = F.relu(conv(h, edge_index))

        probs = []
        for i, j in protein_pairs:
            pair_features = torch.cat([h[i], h[j]], dim=-1)
            prob = self.predictor(pair_features)
            probs.append(prob.item())

        return probs
```

### 算法 4.3.2 (基因调控网络建模算法 / Gene Regulatory Network Modeling Algorithm)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GCNConv
from typing import Dict, List, Tuple

class GeneRegulatoryNetwork(nn.Module):
    """基因调控网络建模 - 使用自适应GNN"""

    def __init__(self, gene_dim: int = 64, hidden_dim: int = 32,
                 num_regulatory_types: int = 2):
        super(GeneRegulatoryNetwork, self).__init__()

        # 编码器：将基因特征编码为隐藏表示
        self.encoder = nn.Sequential(
            nn.Linear(gene_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim)
        )

        # 图卷积层：学习基因间的调控关系
        self.conv1 = GCNConv(hidden_dim, hidden_dim)
        self.conv2 = GCNConv(hidden_dim, hidden_dim)

        # 调控类型预测器：预测激活或抑制
        self.regulatory_type_predictor = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, num_regulatory_types),
            nn.Softmax(dim=-1)
        )

        # 调控强度预测器
        self.strength_predictor = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1),
            nn.Tanh()  # 输出范围 [-1, 1]
        )

    def forward(self, x, edge_index):
        """
        前向传播

        Args:
            x: 基因特征矩阵 [N, gene_dim]
            edge_index: 调控关系边索引 [2, E]

        Returns:
            regulatory_types: 调控类型 [E, num_regulatory_types]
            strengths: 调控强度 [E, 1]
        """
        # 编码基因特征
        h = self.encoder(x)

        # 图卷积
        h = F.relu(self.conv1(h, edge_index))
        h = self.conv2(h, edge_index)

        # 预测调控类型和强度
        src, dst = edge_index
        pair_features = torch.cat([h[src], h[dst]], dim=-1)

        regulatory_types = self.regulatory_type_predictor(pair_features)
        strengths = self.strength_predictor(pair_features)

        return regulatory_types, strengths

    def predict_expression(self, x, edge_index, initial_expression):
        """
        预测基因表达水平

        Args:
            x: 基因特征矩阵
            edge_index: 调控关系边索引
            initial_expression: 初始表达水平 [N, 1]

        Returns:
            predicted_expression: 预测的表达水平 [N, 1]
        """
        regulatory_types, strengths = self.forward(x, edge_index)

        # 构建调控矩阵
        N = x.size(0)
        regulatory_matrix = torch.zeros(N, N)

        src, dst = edge_index
        for k, (i, j) in enumerate(zip(src, dst)):
            # 激活或抑制
            reg_type = regulatory_types[k].argmax().item()
            strength = strengths[k].item()

            if reg_type == 0:  # 激活
                regulatory_matrix[i, j] = strength
            else:  # 抑制
                regulatory_matrix[i, j] = -strength

        # 预测表达水平（简化：线性组合）
        predicted_expression = initial_expression + regulatory_matrix @ initial_expression

        return predicted_expression
```

### 算法 4.3.3 (药物-靶点相互作用预测算法 / Drug-Target Interaction Prediction Algorithm)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GATConv
from typing import List, Tuple

class DrugTargetInteractionPredictor(nn.Module):
    """药物-靶点相互作用预测器"""

    def __init__(self, drug_dim: int = 256, target_dim: int = 128,
                 hidden_dim: int = 64, num_heads: int = 8):
        super(DrugTargetInteractionPredictor, self).__init__()

        # 药物编码器
        self.drug_encoder = nn.Sequential(
            nn.Linear(drug_dim, hidden_dim * 2),
            nn.ReLU(),
            nn.Linear(hidden_dim * 2, hidden_dim)
        )

        # 靶点编码器
        self.target_encoder = nn.Sequential(
            nn.Linear(target_dim, hidden_dim * 2),
            nn.ReLU(),
            nn.Linear(hidden_dim * 2, hidden_dim)
        )

        # 图注意力层（用于学习药物-靶点网络中的关系）
        self.gat = GATConv(hidden_dim, hidden_dim, heads=num_heads,
                          concat=False, dropout=0.2)

        # 相互作用预测器
        self.interaction_predictor = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1),
            nn.Sigmoid()
        )

    def forward(self, drug_features, target_features,
                drug_target_graph=None, drug_target_pairs: List[Tuple[int, int]] = None):
        """
        前向传播

        Args:
            drug_features: 药物特征 [N_drug, drug_dim]
            target_features: 靶点特征 [N_target, target_dim]
            drug_target_graph: 药物-靶点图（可选）
            drug_target_pairs: 要预测的药物-靶点对列表

        Returns:
            interaction_probs: 相互作用概率
        """
        # 编码药物和靶点特征
        drug_emb = self.drug_encoder(drug_features)
        target_emb = self.target_encoder(target_features)

        # 如果提供了图结构，使用GAT学习更好的表示
        if drug_target_graph is not None:
            # 合并药物和靶点嵌入
            combined_emb = torch.cat([drug_emb, target_emb], dim=0)
            combined_emb = self.gat(combined_emb, drug_target_graph.edge_index)
            drug_emb = combined_emb[:drug_emb.size(0)]
            target_emb = combined_emb[drug_emb.size(0):]

        # 预测相互作用
        if drug_target_pairs is None:
            # 预测所有可能的对
            interactions = []
            for i in range(drug_emb.size(0)):
                for j in range(target_emb.size(0)):
                    pair_features = torch.cat([drug_emb[i], target_emb[j]], dim=-1)
                    prob = self.interaction_predictor(pair_features)
                    interactions.append(prob)
            return torch.stack(interactions) if interactions else torch.tensor([])
        else:
            # 预测指定的对
            interactions = []
            for drug_idx, target_idx in drug_target_pairs:
                pair_features = torch.cat([drug_emb[drug_idx], target_emb[target_idx]], dim=-1)
                prob = self.interaction_predictor(pair_features)
                interactions.append(prob)
            return torch.stack(interactions)
```

## 📊 **性能评估 / Performance Evaluation**

### 4.3.1 评估指标 / Evaluation Metrics

#### 预测准确率指标

1. **蛋白质相互作用预测**:
   - **准确率 (Accuracy)**: $\text{Acc} = \frac{TP + TN}{TP + TN + FP + FN}$
   - **精确率 (Precision)**: $\text{Prec} = \frac{TP}{TP + FP}$
   - **召回率 (Recall)**: $\text{Rec} = \frac{TP}{TP + FN}$
   - **F1分数**: $\text{F1} = \frac{2 \cdot \text{Prec} \cdot \text{Rec}}{\text{Prec} + \text{Rec}}$
   - **AUC-ROC**: 受试者工作特征曲线下面积
   - **AUC-PR**: 精确率-召回率曲线下面积

2. **基因表达预测**:
   - **均方误差 (MSE)**: $\text{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$
   - **平均绝对误差 (MAE)**: $\text{MAE} = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|$
   - **相关系数 (Pearson)**: $r = \frac{\sum_{i}(y_i - \bar{y})(\hat{y}_i - \bar{\hat{y}})}{\sqrt{\sum_{i}(y_i - \bar{y})^2}\sqrt{\sum_{i}(\hat{y}_i - \bar{\hat{y}})^2}}$

3. **药物-靶点相互作用预测**:
   - **AUROC**: 通常 > 0.85 为优秀
   - **AUPRC**: 通常 > 0.80 为优秀
   - **Top-K准确率**: 前K个预测中正确预测的比例

#### 系统性能指标

1. **计算效率**:
   - **训练时间**: 模型训练所需时间
   - **推理时间**: 单次预测所需时间
   - **吞吐量**: 每秒处理的样本数

2. **内存使用**:
   - **模型大小**: 参数量
   - **内存占用**: 训练和推理时的内存使用

3. **可扩展性**:
   - **网络规模**: 能处理的网络节点数上限
   - **批处理能力**: 能同时处理的样本数

### 4.3.3 复杂度分析 / Complexity Analysis

#### 时间复杂度

1. **蛋白质相互作用预测**:
   - 图卷积: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d))$，其中 $L$ 是层数，$d$ 是特征维度
   - 相互作用预测: $O(|V|^2 \cdot d)$
   - **总复杂度**: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d) + |V|^2 \cdot d)$

2. **基因调控网络建模**:
   - 图卷积: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d))$
   - 调控预测: $O(|E| \cdot d)$
   - **总复杂度**: $O(L \cdot (|V| \cdot d^2 + |E| \cdot d))$

#### 空间复杂度

- **模型参数**: $O(L \cdot d^2)$
- **中间表示**: $O(|V| \cdot d + |E|)$
- **总空间复杂度**: $O(L \cdot d^2 + |V| \cdot d + |E|)$
