## 📐 **形式化定义 / Formal Definition**

### 定义 5.2 (复杂系统控制问题 / Complex System Control Problem)

**复杂系统控制问题**是在满足约束条件下，通过控制输入使系统达到期望状态：

$$\min_{u(\cdot)} J(x(\cdot), u(\cdot)) = \int_0^T L(x(t), u(t), t) dt + \Phi(x(T))$$

subject to:
$$\dot{x}(t) = f(x(t), u(t), t, w(t)), \quad x(0) = x_0$$
$$g(x(t), u(t), t) \leq 0, \quad \forall t \in [0, T]$$
$$h(x(T)) = 0$$
$$u(t) \in \mathcal{U}, \quad x(t) \in \mathcal{X}$$

其中：

- $x(t) \in \mathbb{R}^n$ 是系统状态向量
- $u(t) \in \mathbb{R}^m$ 是控制输入向量
- $w(t) \in \mathbb{R}^p$ 是扰动/不确定性
- $f: \mathbb{R}^n \times \mathbb{R}^m \times \mathbb{R} \times \mathbb{R}^p \to \mathbb{R}^n$ 是系统动力学函数
- $L: \mathbb{R}^n \times \mathbb{R}^m \times \mathbb{R} \to \mathbb{R}$ 是运行成本函数
- $\Phi: \mathbb{R}^n \to \mathbb{R}$ 是终端成本函数
- $g, h$ 是约束函数
- $\mathcal{U}, \mathcal{X}$ 是控制和状态约束集合
- $T$ 是时间范围（可能为 $\infty$，无限时域控制）

**控制目标类型**:

1. **轨迹跟踪**: 使 $x(t)$ 跟踪期望轨迹 $x_d(t)$
2. **镇定控制**: 使系统稳定到平衡点 $x^*$
3. **最优控制**: 最小化性能指标 $J$
4. **鲁棒控制**: 在不确定性 $w(t)$ 下保证性能

### 定义 5.2.1 (可控性 / Controllability)

对于线性时不变系统：
$$\dot{x}(t) = Ax(t) + Bu(t)$$

系统是**可控的**（controllable），如果可控性矩阵：
$$C = [B, AB, A^2B, \ldots, A^{n-1}B]$$

满秩，即 $\text{rank}(C) = n$。

**可控性的直观理解**: 存在控制输入 $u(t)$，可以在有限时间内将系统从任意初始状态 $x(0)$ 驱动到任意目标状态 $x(T)$。

对于非线性系统，可控性是局部的，通常通过Lie代数条件判断。

### 定义 5.2.2 (可观性 / Observability)

对于线性时不变系统：
$$\begin{cases}
\dot{x}(t) = Ax(t) + Bu(t) \\
y(t) = Cx(t)
\end{cases}$$

系统是**可观的**（observable），如果可观性矩阵：
$$O = \begin{bmatrix} C \\ CA \\ CA^2 \\ \vdots \\ CA^{n-1} \end{bmatrix}$$

满秩，即 $\text{rank}(O) = n$。

**可观性的直观理解**: 可以通过输出 $y(t)$ 和输入 $u(t)$ 的历史信息，唯一确定系统的初始状态 $x(0)$。

### 定义 5.2.3 (稳定性 / Stability)

系统在平衡点 $x^*$ 处（满足 $f(x^*, 0, t) = 0$）是：

1. **Lyapunov稳定**: 对于任意 $\epsilon > 0$，存在 $\delta > 0$，使得 $\|x(0) - x^*\| < \delta \implies \|x(t) - x^*\| < \epsilon, \forall t \geq 0$

2. **渐近稳定**: 系统是Lyapunov稳定的，且 $\lim_{t \to \infty} x(t) = x^*$

3. **指数稳定**: 存在 $\alpha, \beta > 0$，使得 $\|x(t) - x^*\| \leq \alpha \|x(0) - x^*\| e^{-\beta t}$

## 🔧 **控制方法类型 / Types of Control Methods**

### 1. 多尺度控制 / Multi-Scale Control

**方法原理**:
多尺度控制在多个时间/空间尺度上分别设计控制器，然后通过协调机制实现全局控制目标。

**控制律形式**:
$$u(t) = \sum_{i=1}^n w_i u_i(t, s_i)$$

其中 $u_i(t, s_i)$ 是尺度 $s_i$ 上的局部控制器，$w_i$ 是权重。

**协调机制**:
- **层次化协调**: 粗尺度控制器设定目标，细尺度控制器执行
- **并行协调**: 各尺度控制器并行工作，通过信息交换协调
- **自适应协调**: 根据系统状态动态调整各尺度的权重

**优势**:
- 充分利用各尺度的特性
- 降低计算复杂度
- 提高控制的鲁棒性

**挑战**:
- 尺度间耦合的处理
- 协调机制的设计
- 稳定性分析

**应用**: 交通系统控制、生态系统管理、多尺度制造系统

### 2. 自适应控制 / Adaptive Control

**方法原理**:
自适应控制根据系统状态和性能指标，动态调整控制器参数，适应系统参数变化和未知扰动。

**自适应律**:
$$\dot{\theta}(t) = -\Gamma \phi(x(t), e(t)) e(t)$$

其中：
- $\theta(t)$ 是自适应参数
- $\Gamma > 0$ 是学习率矩阵
- $\phi$ 是回归向量
- $e(t)$ 是跟踪误差

**自适应控制类型**:

1. **模型参考自适应控制（MRAC）**:
   - 参考模型: $\dot{x}_m(t) = A_m x_m(t) + B_m r(t)$
   - 控制律: $u(t) = K_x(t)x(t) + K_r(t)r(t)$
   - 参数更新: $\dot{K}_x(t) = -\Gamma_x x(t) e^T(t) PB$

2. **自校正控制（STC）**:
   - 在线参数估计 + 控制器重新设计
   - 参数估计: $\hat{\theta}(t) = \arg\min_\theta \sum_{i=1}^t \|y(i) - \phi^T(i)\theta\|^2$
   - 控制器设计: 基于估计参数设计控制器

3. **直接自适应控制**:
   - 直接调整控制器参数，无需参数估计

**优势**:
- 适应系统参数变化
- 对扰动和不确定性鲁棒
- 无需精确系统模型

**挑战**:
- 稳定性保证（需要满足持续性激励条件）
- 参数收敛性
- 瞬态性能

**应用**: 工业过程控制、机器人控制、航空航天系统、电力系统

### 3. 分布式控制 / Distributed Control

**方法原理**:
分布式控制将系统分解为多个子系统，每个子系统有局部控制器，通过通信网络协调实现全局控制目标。

**控制架构**:
- **完全分布式**: 每个节点只与邻居节点通信
- **层次化分布式**: 存在层次结构，高层节点协调低层节点
- **混合架构**: 结合集中式和分布式控制

**控制律**:
$$u_i(t) = K_i x_i(t) + \sum_{j \in N(i)} K_{ij} x_j(t-\tau_{ij})$$

其中 $N(i)$ 是节点 $i$ 的邻居集合，$\tau_{ij}$ 是通信延迟。

**一致性控制**:
$$\lim_{t \to \infty} \|x_i(t) - x_j(t)\| = 0, \quad \forall i, j$$

**优势**:
- 可扩展性好
- 容错性强（单点故障不影响全局）
- 降低通信和计算负担

**挑战**:
- 通信延迟和丢包
- 网络拓扑变化
- 全局性能保证

**应用**: 智能电网、多机器人系统、传感器网络、无人机编队

### 4. 最优控制 / Optimal Control

**方法原理**:
最优控制使用变分法或动态规划，求解使性能指标最小的控制输入。

**线性二次调节器（LQR）**:
对于系统 $\dot{x} = Ax + Bu$，目标函数：
$$J = \int_0^\infty (x^T Q x + u^T R u) dt$$

最优控制律：
$$u^*(t) = -R^{-1}B^T P x(t) = -K x(t)$$

其中 $P$ 是Riccati方程的解：
$$A^T P + PA - PBR^{-1}B^T P + Q = 0$$

**动态规划（离散时间）**:
Bellman方程：
$$V^*(x_k) = \min_{u_k} [L(x_k, u_k) + V^*(f(x_k, u_k))]$$

**Pontryagin极大值原理（连续时间）**:
$$H(x^*, u^*, \lambda^*, t) = \max_{u \in \mathcal{U}} H(x^*, u, \lambda^*, t)$$

其中Hamiltonian函数：
$$H(x, u, \lambda, t) = L(x, u, t) + \lambda^T f(x, u, t)$$

**优势**:
- 理论上最优
- 系统性能可预测
- 成熟的数学理论

**挑战**:
- 需要精确系统模型
- 计算复杂度高（对于非线性系统）
- 实时性限制

**应用**: 航空航天、经济系统、化工过程、机器人轨迹规划

### 5. 鲁棒控制 / Robust Control

**方法原理**:
鲁棒控制在存在不确定性和扰动的情况下，保证系统稳定性和性能。

**$H_\infty$ 控制**:
最小化从扰动 $w$ 到输出 $z$ 的 $H_\infty$ 范数：
$$\|T_{zw}\|_\infty = \sup_{\omega} \bar{\sigma}(T_{zw}(j\omega)) < \gamma$$

其中 $\gamma > 0$ 是性能水平，$\bar{\sigma}$ 是最大奇异值。

**$\mu$ 综合**:
处理结构化不确定性，求解：
$$\min_K \sup_{\Delta \in \Delta} \mu(M(K, \Delta))$$

**滑模控制**:
设计滑动面 $s(x) = 0$，使系统状态在有限时间内到达滑动面，并在其上滑动到平衡点。

控制律：
$$u(t) = u_{eq}(t) + K \cdot \text{sign}(s(x))$$

**优势**:
- 对不确定性鲁棒
- 性能界限可保证
- 设计方法系统化

**挑战**:
- 保守性（可能过于保守）
- 计算复杂度
- 实际应用中的权衡

**应用**: 航空航天、电力系统、机械系统、化工过程

### 6. 预测控制 / Predictive Control

**方法原理**:
模型预测控制（MPC）在每个采样时刻，基于当前状态和预测模型，求解有限时域优化问题，只执行第一步控制，然后重新优化。

**优化问题**:
$$\min_{u_{k:k+N-1}} \sum_{i=0}^{N-1} L(x_{k+i}, u_{k+i}) + \Phi(x_{k+N})$$

subject to:
$$x_{k+i+1} = f(x_{k+i}, u_{k+i})$$
$$x_{k+i} \in \mathcal{X}, \quad u_{k+i} \in \mathcal{U}$$
$$x_{k+N} \in \mathcal{X}_f$$

**特点**:
- 显式处理约束
- 多变量控制
- 适用于非线性系统

**优势**:
- 能处理约束
- 性能优异
- 易于理解和调试

**挑战**:
- 计算负担重
- 稳定性保证（需要终端约束/代价）
- 对模型误差敏感

**应用**: 化工过程、电力系统、交通系统、机器人控制

### 7. 学习控制 / Learning Control

**方法原理**:
学习控制通过从数据中学习，不断改进控制性能，无需精确系统模型。

**强化学习控制**:
- **Q学习**: 学习最优动作值函数 $Q^*(s, a)$
- **策略梯度**: 直接优化策略 $\pi(a|s; \theta)$
- **Actor-Critic**: 结合值函数和策略学习

**迭代学习控制（ILC）**:
$$u_{k+1}(t) = u_k(t) + L \cdot e_k(t)$$

其中 $e_k(t) = y_d(t) - y_k(t)$ 是跟踪误差。

**神经网络控制**:
使用神经网络逼近非线性函数：
$$u(t) = N(x(t), \theta)$$

参数通过反向传播或强化学习更新。

**优势**:
- 无需精确模型
- 能处理复杂非线性
- 自适应性强

**挑战**:
- 需要大量数据
- 稳定性保证困难
- 可解释性差

**应用**: 机器人控制、自动驾驶、游戏AI、工业自动化

### 算法 5.2.1 (LQR最优控制算法)

```python
import numpy as np
from scipy.linalg import solve_continuous_are
from typing import Tuple

def lqr_control(A: np.ndarray, B: np.ndarray, Q: np.ndarray,
                R: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
    """
    线性二次调节器（LQR）控制

    参数:
        A: 系统矩阵 (n x n)
        B: 输入矩阵 (n x m)
        Q: 状态权重矩阵 (n x n)，半正定
        R: 控制权重矩阵 (m x m)，正定

    返回:
        K: 反馈增益矩阵 (m x n)
        P: Riccati方程的解 (n x n)
    """
    # 求解连续时间代数Riccati方程
    P = solve_continuous_are(A, B, Q, R)

    # 计算反馈增益
    K = np.linalg.solve(R, B.T @ P)

    return K, P

# 使用示例
A = np.array([[0, 1], [-1, -1]])
B = np.array([[0], [1]])
Q = np.eye(2)
R = np.array([[1]])

K, P = lqr_control(A, B, Q, R)
print(f"反馈增益 K = {K}")
```

### 算法 5.2.2 (模型预测控制算法)

```python
from scipy.optimize import minimize
import numpy as np

class MPCController:
    """模型预测控制器"""

    def __init__(self, A: np.ndarray, B: np.ndarray, Q: np.ndarray,
                 R: np.ndarray, N: int, umax: float = None):
        self.A = A
        self.B = B
        self.Q = Q
        self.R = R
        self.N = N  # 预测时域
        self.umax = umax

    def control(self, x0: np.ndarray, x_ref: np.ndarray) -> np.ndarray:
        """计算MPC控制输入"""
        n, m = self.A.shape[0], self.B.shape[1]

        def cost(u_sequence):
            """预测时域内的成本"""
            u_seq = u_sequence.reshape(self.N, m)
            x = x0.copy()
            cost_val = 0.0

            for i in range(self.N):
                u = u_seq[i]
                # 状态演化
                x = self.A @ x + self.B @ u
                # 状态成本
                cost_val += (x - x_ref).T @ self.Q @ (x - x_ref)
                # 控制成本
                cost_val += u.T @ self.R @ u

            return cost_val

        # 约束
        constraints = []
        if self.umax is not None:
            constraints.append({'type': 'ineq',
                              'fun': lambda u: self.umax - np.abs(u)})

        # 优化
        u0 = np.zeros(self.N * m)
        result = minimize(cost, u0, method='SLSQP', constraints=constraints)

        # 返回第一步控制（Receding Horizon）
        u_opt = result.x.reshape(self.N, m)
        return u_opt[0]
```

### 算法 5.2.3 (自适应控制算法 - MRAC)

```python
class MRACController:
    """模型参考自适应控制器"""

    def __init__(self, Am: np.ndarray, Bm: np.ndarray,
                 Gamma_x: float, Gamma_r: float):
        self.Am = Am  # 参考模型
        self.Bm = Bm
        self.Gamma_x = Gamma_x  # 自适应增益
        self.Gamma_r = Gamma_r
        self.Kx = np.zeros((Bm.shape[1], Am.shape[0]))
        self.Kr = np.eye(Bm.shape[1])

    def update(self, x: np.ndarray, xm: np.ndarray, r: np.ndarray,
               dt: float) -> np.ndarray:
        """更新控制器参数并计算控制输入"""
        # 跟踪误差
        e = xm - x

        # 参数更新（梯度下降）
        self.Kx += self.Gamma_x * np.outer(e, x) * dt
        self.Kr += self.Gamma_r * np.outer(e, r) * dt

        # 控制输入
        u = self.Kx @ x + self.Kr @ r

        return u
```

## 📊 **复杂度分析 / Complexity Analysis**

### 时间复杂度

**LQR控制**:
- Riccati方程求解: $O(N^3)$ 其中 $N$ 是状态维度
- 在线计算: $O(M \cdot N)$ 其中 $M$ 是控制维度
- **总复杂度**: $O(N^3)$（离线） + $O(M \cdot N)$（在线）

**MPC控制**:
- 每次优化: $O(T \cdot (N^3 + M^3))$ 其中 $T$ 是预测时域
- 约束处理: $O(T \cdot (N + M))$
- **总复杂度**: $O(T \cdot (N^3 + M^3))$ 每次采样

**自适应控制**:
- 参数更新: $O(P \cdot N)$ 其中 $P$ 是参数数量
- 控制计算: $O(M \cdot N)$
- **总复杂度**: $O(P \cdot N + M \cdot N)$ 每次采样

**分布式控制**:
- 局部控制: $O(N_i^2)$ 其中 $N_i$ 是局部状态维度
- 通信开销: $O(D \cdot N)$ 其中 $D$ 是通信延迟
- **总复杂度**: $O(\max_i N_i^2 + D \cdot N)$

### 空间复杂度

- **LQR**: $O(N^2 + M \cdot N)$
- **MPC**: $O(T \cdot (N + M))$
- **自适应控制**: $O(P + N + M)$
- **分布式控制**: $O(\sum_i N_i^2)$

## 🔬 **理论分析 / Theoretical Analysis**

### 定理 5.2.1 (可控性判据 / Controllability Criterion)

**定理**: 线性时不变系统 $\dot{x} = Ax + Bu$ 可控，当且仅当可控性矩阵 $C = [B, AB, \ldots, A^{n-1}B]$ 满秩。

**证明**: 使用Cayley-Hamilton定理和状态可达性分析。

### 定理 5.2.2 (LQR稳定性定理 / LQR Stability Theorem)

**定理**: 如果系统 $(A, B)$ 可控，$(A, Q^{1/2})$ 可观，$Q \succeq 0$，$R \succ 0$，则LQR控制器使闭环系统渐近稳定。

**证明**: 使用Lyapunov函数 $V(x) = x^T P x$，其中 $P$ 是Riccati方程的解。
