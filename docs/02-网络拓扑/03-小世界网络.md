# 3. 小世界网络

## 3.1 基本定义

- **定义 3.1.1** 小世界网络是指平均路径长度较短且聚类系数较高的网络结构。
- **定义 3.1.2** Watts-Strogatz模型：通过随机重连规则生成小世界网络。

## 3.2 结构特征

- 高聚类系数
- 短平均路径长度
- 介于规则网络与随机网络之间

## 3.3 重要定理

- **定理 3.3.1**（小世界效应）：在一定重连概率下，网络平均路径长度急剧下降而聚类系数保持较高。
- **定理 3.3.2**（六度分隔）：任意两节点间的平均距离随网络规模对数增长。
