## 1. 小世界网络基础

### 1.1 基本定义

**定义 1.1** (小世界网络)
**小世界网络**是具有短平均路径长度和高聚类系数的网络：
$$\mathcal{SW} = \langle G, L, C, \gamma \rangle$$

其中：

- $G = (V, E)$ 是图结构
- $L$ 是平均路径长度
- $C$ 是聚类系数
- $\gamma$ 是度分布指数

**定义 1.2** (小世界特性)
**小世界特性**满足：
$$L \sim \log N \quad \text{且} \quad C \gg C_{random}$$

其中 $N$ 是节点数，$C_{random}$ 是随机网络的聚类系数。

### 1.2 结构特征

**定义 1.3** (聚类系数)
**聚类系数**衡量网络的局部聚集程度：
$$C = \frac{1}{N} \sum_{i} C_i$$

其中 $C_i$ 是节点 $i$ 的局部聚类系数：
$$C_i = \frac{2E_i}{k_i(k_i - 1)}$$

**定义 1.4** (平均路径长度)
**平均路径长度**是网络中任意两节点间最短路径的平均长度：
$$L = \frac{1}{N(N-1)} \sum_{i \neq j} d_{ij}$$

其中 $d_{ij}$ 是节点 $i$ 和 $j$ 之间的距离。

## 2. Watts-Strogatz模型

### 2.0 小世界网络模型对比矩阵 / Small World Network Models Comparison Matrix

| 网络模型 | 生成方式 | 聚类系数 | 路径长度 | 度分布 | 优点 | 缺点 | 典型应用 |
|---------|---------|---------|---------|--------|------|------|---------|
| **Watts-Strogatz** | 重连规则 | 高 | 短 | 均匀 | 简单、可控 | 度分布均匀 | 社交网络 |
| **Newman-Watts** | 添加边 | 高 | 短 | 均匀 | 保持连通性 | 可能增加边数 | 通信网络 |
| **Kleinberg模型** | 层次结构 | 中 | 短 | 幂律 | 可导航 | 实现复杂 | 信息检索 |
| **随机网络** | 随机连接 | 低 | 短 | 泊松 | 简单 | 无聚类 | 理论分析 |
| **规则网络** | 规则连接 | 高 | 长 | 均匀 | 高聚类 | 路径长 | 局部网络 |
| **无标度网络** | 优先连接 | 低 | 短 | 幂律 | 真实网络 | 低聚类 | 互联网 |

### 2.1 模型定义

**定义 2.1** (Watts-Strogatz模型)
**Watts-Strogatz模型**通过重连规则生成小世界网络：
$$\mathcal{WS}(N, k, p) = \langle V, E, p \rangle$$

其中：

- $N$ 是节点数
- $k$ 是初始每个节点的邻居数
- $p$ 是重连概率

**算法 2.1** (Watts-Strogatz算法)

```text
输入：节点数 N，邻居数 k，重连概率 p
输出：小世界网络 G

1. 构建环形网络：for i = 1 to N do
   a. for j = 1 to k/2 do
      b. 连接节点 i 和 (i+j) mod N
2. 随机重连：for each edge (i,j) do
   a. if random() < p then
      b. 选择新节点 k ≠ i,j
      c. 重连边 (i,k)
3. 返回网络：return G
```

### 2.2 模型特性

**定理 2.1** (小世界效应)
当重连概率 $p$ 在中间范围时，网络具有小世界特性：
$$L(p) \sim \log N \quad \text{且} \quad C(p) \gg C_{random}$$

**证明：**

1. 当 $p = 0$ 时，网络为规则网络，$L \sim N$，$C$ 较高
2. 当 $p = 1$ 时，网络为随机网络，$L \sim \log N$，$C$ 较低
3. 当 $0 < p < 1$ 时，通过重连引入长距离连接，保持局部聚集性
$\boxed{\text{证毕}}$

## 3. 小世界网络分析

### 3.1 度分布

**定义 3.1** (度分布)
**度分布**描述网络中节点的连接数分布：
$$P(k) = \frac{|\{v : \deg(v) = k\}|}{N}$$

**定理 3.1** (度分布特性)
小世界网络的度分布近似为泊松分布：
$$P(k) \approx \frac{\lambda^k e^{-\lambda}}{k!}$$

其中 $\lambda = \langle k \rangle$ 是平均度。

### 3.2 连通性分析

**定义 3.2** (连通分量)
**连通分量**是网络中最大的连通子图：
$$CC(G) = \{C_1, C_2, \ldots, C_k\}$$

**定理 3.2** (连通性)
小世界网络通常是连通的，即 $|CC(G)| = 1$。

**证明：**
通过重连引入的长距离连接确保网络连通性。
$\boxed{\text{证毕}}$

### 3.3 中心性分析

**定义 3.3** (度中心性)
**度中心性**衡量节点的直接连接数：
$$C_D(i) = \frac{k_i}{N-1}$$

**定义 3.4** (接近中心性)
**接近中心性**衡量节点到其他节点的平均距离：
$$C_C(i) = \frac{N-1}{\sum_{j \neq i} d_{ij}}$$

**算法 3.1** (中心性计算)

```text
输入：小世界网络 G
输出：中心性指标

1. 计算度中心性：for each v in V do
   a. C_D(v) = degree(v) / (N - 1)
2. 计算接近中心性：for each v in V do
   a. C_C(v) = (N - 1) / sum(shortest_paths(v))
3. 返回中心性：return {C_D, C_C}
```

## 4. 小世界网络应用

### 4.1 信息传播

**定义 4.1** (传播模型)
**传播模型**描述信息在小世界网络中的传播：
$$\frac{dI_i(t)}{dt} = \beta \sum_{j} A_{ij} I_j(t) - \gamma I_i(t)$$

其中 $I_i(t)$ 是节点 $i$ 在时刻 $t$ 的感染状态。

**定理 4.1** (传播效率)
小世界网络中的信息传播效率高于规则网络和随机网络。

**证明：**
短平均路径长度确保快速传播，高聚类系数确保局部聚集。
$\boxed{\text{证毕}}$

### 4.2 同步化

**定义 4.2** (同步化模型)
**同步化模型**描述节点状态在小世界网络中的同步：
$$\frac{dx_i(t)}{dt} = f(x_i(t)) + \sigma \sum_{j} A_{ij} (x_j(t) - x_i(t))$$

其中 $\sigma$ 是耦合强度。

**定理 4.2** (同步化能力)
小世界网络具有较强的同步化能力。

**证明：**
短平均路径长度减少同步时间，高聚类系数增强局部同步。
$\boxed{\text{证毕}}$

## 5. 小世界网络生成算法

### 5.1 基础算法

**算法 5.1** (基础小世界生成)

```text
输入：节点数 N，邻居数 k，重连概率 p
输出：小世界网络 G

1. 初始化环形网络：G = create_ring_network(N, k)
2. 随机重连：for each edge e in G.edges do
   a. if random() < p then
      b. 选择新节点 v ≠ e.endpoints
      c. 重连边 e 到 v
3. 返回网络：return G
```

### 5.2 改进算法

**算法 5.2** (改进小世界生成)

```text
输入：节点数 N，目标聚类系数 C_target，目标路径长度 L_target
输出：小世界网络 G

1. 初始化参数：p = 0.1
2. 迭代优化：while not converged do
   a. 生成网络：G = generate_ws_network(N, k, p)
   b. 计算指标：C = compute_clustering(G), L = compute_path_length(G)
   c. 调整参数：p = adjust_parameter(p, C, L, C_target, L_target)
3. 返回最优网络：return G
```

## 6. 小世界网络验证

### 6.1 统计验证

**定义 6.1** (小世界指数)
**小世界指数**衡量网络的小世界特性：
$$\sigma = \frac{C/C_{random}}{L/L_{random}}$$

其中 $C_{random}$ 和 $L_{random}$ 是随机网络的对应指标。

**算法 6.1** (小世界验证)

```text
输入：网络 G
输出：小世界指数 σ

1. 计算指标：C = clustering_coefficient(G), L = average_path_length(G)
2. 生成随机网络：G_random = random_network(G)
3. 计算随机指标：C_random = clustering_coefficient(G_random)
4. 计算指数：σ = (C/C_random) / (L/L_random)
5. 返回指数：return σ
```

## 7. 小世界网络的结构化梳理、主要定理、极值、语义模型与自动化验证

### 7.2 主要定理与极值

**定理 7.2.1（小世界极值）**
最优重连概率使传播效率最大化，同时保持高聚类系数。

**定理 7.2.2（同步化极值）**
小世界网络在特定耦合强度下达到最优同步化效果。

**定理 7.2.3（连通性极值）**
最小重连概率确保网络连通性的同时最大化聚类系数。

### 7.3 形式语义模型

- 小世界网络$SW=(G,L,C,\gamma)$，重连概率$p$，传播模型$P$，同步化模型$S$等
- 性质可用一阶/概率/时序逻辑公式表达，如$\forall i,j\in V, d(i,j)\leq\log N$（短路径性）
- 小世界生成、传播分析、同步化等可形式化为范畴上的函子或逻辑推理过程

### 7.4 保持性与极值定理

**定理 7.4.1（小世界映射下结构保持）**
小世界映射$h:SW_1\to SW_2$保持短路径性、高聚类性等结构性性质。

**定理 7.4.2（极值保持性）**
最优传播效率、同步化效果等极值性质在小世界保持映射下不减弱。

**算法 9.1.1** (多层小世界网络生成 / Multilayer Small World Network Generation)

```python
import networkx as nx
import numpy as np

class MultilayerSmallWorld:
    """多层小世界网络"""

    def __init__(self, num_layers, num_nodes, k, p):
        self.num_layers = num_layers
        self.num_nodes = num_nodes
        self.k = k  # 每层邻居数
        self.p = p  # 重连概率
        self.layers = []
        self.inter_layer_edges = []

    def generate_multilayer_network(self):
        """生成多层小世界网络"""
        # 为每一层生成小世界网络
        for layer_id in range(self.num_layers):
            layer = self.generate_single_layer(layer_id)
            self.layers.append(layer)

        # 添加层间连接
        self.add_inter_layer_connections()

        return self.layers, self.inter_layer_edges

    def generate_single_layer(self, layer_id):
        """生成单层小世界网络"""
        # 使用Watts-Strogatz模型
        G = nx.watts_strogatz_graph(self.num_nodes, self.k, self.p)

        # 添加层标识
        for node in G.nodes():
            G.nodes[node]['layer'] = layer_id

        return G

    def add_inter_layer_connections(self, inter_layer_prob=0.1):
        """添加层间连接"""
        for i in range(self.num_layers):
            for j in range(i + 1, self.num_layers):
                # 在层i和层j之间添加连接
                for node_i in self.layers[i].nodes():
                    for node_j in self.layers[j].nodes():
                        if np.random.random() < inter_layer_prob:
                            self.inter_layer_edges.append(
                                ((i, node_i), (j, node_j))
                            )

    def compute_multilayer_clustering(self):
        """计算多层聚类系数"""
        clustering_coefficients = []

        for layer in self.layers:
            clustering = nx.average_clustering(layer)
            clustering_coefficients.append(clustering)

        # 考虑层间连接的影响
        inter_layer_clustering = self.compute_inter_layer_clustering()

        return {
            'intra_layer': clustering_coefficients,
            'inter_layer': inter_layer_clustering,
            'overall': np.mean(clustering_coefficients)
        }

    def compute_inter_layer_clustering(self):
        """计算层间聚类"""
        # 简化版本：计算跨层三角形的数量
        inter_layer_triangles = 0
        # 实际实现需要更复杂的计算
        return inter_layer_triangles

# 复杂度分析
# 时间复杂度: O(L * N * k + L^2 * N^2) 其中L是层数，N是节点数
# 空间复杂度: O(L * N * k + L^2 * N^2) 存储多层网络
```

**算法 9.2.1** (基于GAN的小世界网络生成 / GAN-based Small World Network Generation)

```python
import torch
import torch.nn as nn
import networkx as nx

class SmallWorldNetworkGAN:
    """基于GAN的小世界网络生成器"""

    def __init__(self, num_nodes, latent_dim=100):
        self.num_nodes = num_nodes
        self.latent_dim = latent_dim
        self.generator = Generator(latent_dim, num_nodes)
        self.discriminator = Discriminator(num_nodes)

    def generate_network(self, z=None):
        """生成小世界网络"""
        if z is None:
            z = torch.randn(1, self.latent_dim)

        # 生成邻接矩阵
        adj_matrix = self.generator(z)

        # 转换为NetworkX图
        G = nx.from_numpy_array(adj_matrix.detach().numpy()[0])

        # 二值化（阈值化）
        G = self.binarize_graph(G, threshold=0.5)

        return G

    def binarize_graph(self, G, threshold=0.5):
        """二值化图（移除低权重边）"""
        edges_to_remove = []
        for u, v, weight in G.edges(data='weight'):
            if weight < threshold:
                edges_to_remove.append((u, v))
        G.remove_edges_from(edges_to_remove)
        return G

class Generator(nn.Module):
    """生成器网络"""

    def __init__(self, latent_dim, num_nodes):
        super(Generator, self).__init__()
        self.num_nodes = num_nodes
        self.fc1 = nn.Linear(latent_dim, 256)
        self.fc2 = nn.Linear(256, 512)
        self.fc3 = nn.Linear(512, num_nodes * num_nodes)
        self.sigmoid = nn.Sigmoid()

    def forward(self, z):
        x = torch.relu(self.fc1(z))
        x = torch.relu(self.fc2(x))
        x = self.fc3(x)
        x = x.view(-1, self.num_nodes, self.num_nodes)
        # 确保对称（无向图）
        x = (x + x.transpose(-2, -1)) / 2
        x = self.sigmoid(x)
        return x

class Discriminator(nn.Module):
    """判别器网络"""

    def __init__(self, num_nodes):
        super(Discriminator, self).__init__()
        self.num_nodes = num_nodes
        self.fc1 = nn.Linear(num_nodes * num_nodes, 512)
        self.fc2 = nn.Linear(512, 256)
        self.fc3 = nn.Linear(256, 1)
        self.sigmoid = nn.Sigmoid()

    def forward(self, adj_matrix):
        x = adj_matrix.view(-1, self.num_nodes * self.num_nodes)
        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        x = self.fc3(x)
        x = self.sigmoid(x)
        return x

# 复杂度分析
# 生成: O(N^2) 生成邻接矩阵
# 训练: O(N^2 * E) 其中E是训练轮数
```

**算法 9.3.1** (小世界神经网络设计 / Small World Neural Network Design)

```python
import torch
import torch.nn as nn

class SmallWorldNeuralNetwork(nn.Module):
    """小世界神经网络"""

    def __init__(self, input_dim, hidden_dim, output_dim, k=4, p=0.1):
        super(SmallWorldNeuralNetwork, self).__init__()
        self.k = k  # 每个节点的邻居数
        self.p = p  # 重连概率

        # 构建小世界连接模式
        self.connections = self.build_small_world_connections(
            hidden_dim, k, p
        )

        # 定义层
        self.input_layer = nn.Linear(input_dim, hidden_dim)
        self.hidden_layers = self.build_hidden_layers(hidden_dim)
        self.output_layer = nn.Linear(hidden_dim, output_dim)

    def build_small_world_connections(self, num_nodes, k, p):
        """构建小世界连接模式"""
        # 创建规则环
        connections = {}
        for i in range(num_nodes):
            neighbors = []
            # 添加k/2个左右邻居
            for j in range(1, k // 2 + 1):
                neighbors.append((i - j) % num_nodes)
                neighbors.append((i + j) % num_nodes)
            connections[i] = neighbors

        # 随机重连
        for i in range(num_nodes):
            for j in range(len(connections[i])):
                if np.random.random() < p:
                    # 重连到随机节点
                    new_neighbor = np.random.randint(0, num_nodes)
                    connections[i][j] = new_neighbor

        return connections

    def build_hidden_layers(self, hidden_dim):
        """构建隐藏层（使用小世界连接）"""
        layers = nn.ModuleList()
        # 使用稀疏连接构建层
        for i in range(hidden_dim):
            # 只连接到小世界邻居
            layer = nn.Linear(len(self.connections[i]), 1)
            layers.append(layer)
        return layers

    def forward(self, x):
        """前向传播"""
        x = torch.relu(self.input_layer(x))

        # 小世界连接的隐藏层
        hidden_outputs = []
        for i, layer in enumerate(self.hidden_layers):
            # 只使用连接的邻居
            neighbor_features = x[:, self.connections[i]]
            output = layer(neighbor_features)
            hidden_outputs.append(output)

        x = torch.cat(hidden_outputs, dim=1)
        x = torch.relu(x)
        x = self.output_layer(x)

        return x

# 复杂度分析
# 参数数量: O(N * k) 其中N是节点数，k是邻居数（远小于全连接O(N^2)）
# 计算复杂度: O(N * k) 每次前向传播
```
