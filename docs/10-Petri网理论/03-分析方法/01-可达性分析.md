# 可达性分析 / Reachability Analysis

## 📚 **概述 / Overview**

可达性分析是检查某个标识是否可以从初始标识到达。这是Petri网分析的基础方法。

---

## 1. 可达性分析 / Reachability Analysis

可达性分析是检查某个标识是否可以从初始标识到达。

### 1.1 可达性问题 / Reachability Problem

**问题**（可达性问题）：给定Petri网 $N$ 和标识 $M$，判断 $M \in R(N, M_0)$。

**复杂度**：可达性问题是 PSPACE-complete 的。

### 1.2 可达性树 / Reachability Tree

可达性树是分析可达性的主要方法，通过系统地探索所有可达标识。
