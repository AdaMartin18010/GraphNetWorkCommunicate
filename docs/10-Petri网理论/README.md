# Petri网理论 / Petri Net Theory

## 📚 **概述 / Overview**

本目录包含Petri网理论的所有相关内容。Petri网（Petri Net）是一种用于描述并发系统、分布式系统和异步系统的数学模型，由 Carl Adam Petri 在1962年提出。
