# 特殊图性质 / Special Graph Properties

## 📚 **概述 / Overview**

本文档描述Petri网具有的特殊图论性质。作为有向二分图，Petri网具有一些特殊的图论性质，这些性质对于理解和分析Petri网非常重要。

---

## 🔧 **核心内容 / Core Content**

- 二分性性质
- 有向性特征
- 权重与多重边
- 图论性质的应用

---

## 📐 **图论性质 / Graph Properties**

Petri网具有以下图论性质：

### 1. 二分性

库所和变迁形成两个不相交的顶点集，所有边都连接这两个集合之间的顶点。

### 2. 有向性

所有边都是有向的，表示从库所到变迁或从变迁到库所的流关系。

### 3. 可能的多重边

如果权重 $W(p, t) > 1$，可以视为多重边，表示一条边对应多个令牌的传输。
