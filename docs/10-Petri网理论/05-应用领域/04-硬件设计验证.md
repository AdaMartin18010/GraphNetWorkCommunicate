- 异步电路设计

## 💼 **应用特点 / Application Features**

Petri网在硬件设计验证中的优势：

1. **异步建模**：适合建模异步电路和异步控制流
2. **并发建模**：可以表示硬件中的并发操作
3. **状态验证**：可以验证硬件状态的正确性
4. **形式化验证**：可以进行硬件设计的自动验证
