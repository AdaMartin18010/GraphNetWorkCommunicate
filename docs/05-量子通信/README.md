### 🔬 **核心概念**

- **量子态**: 量子比特、叠加态、纠缠态
- **量子门**: 单比特门、多比特门、通用量子门
- **量子信道**: 量子噪声、退相干、量子纠错
- **量子密钥分发**: BB84、E91、BBM92协议
- **量子网络**: 量子中继器、量子路由器、量子互联网
- **量子安全**: 信息论安全、设备无关QKD

### 🌐 **重要理论**

- **不可克隆定理**: 量子态无法完美复制
- **贝尔不等式**: 量子纠缠的非局域性
- **量子纠错**: 量子错误纠正码
- **量子信息论**: 量子信道容量、量子纠缠度量
