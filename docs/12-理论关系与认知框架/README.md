# 理论关系与认知框架 / Theory Relationship and Cognitive Framework

## 📚 **模块概述 / Module Overview**

本模块系统梳理Petri网、动态图论、拓扑模型三大理论的内在逻辑脉络，归纳认知规律，建立决策逻辑图谱，提供认知表征工具。

**模块定位**: 理论梳理与认知归纳知识工程
**创建时间**: 2025年1月
**权威对标**: 见 [全项目权威对标来源表-2025-02](../00-项目概述/全项目权威对标来源表-2025-02.md)。本模块综合各模块来源。  
**思维表征与 View 双向绑定**: 见 [View 文件夹思维表征工具集-2025](../../view/View文件夹思维表征工具集-2025.md)、[View 文件夹对比矩阵集-2025](../../view/View文件夹对比矩阵集-2025.md)、[View 文件夹概念关系网络-2025](../../view/View文件夹概念关系网络-2025.md)。本模块 00-理论映射关系图谱、认知表征图谱、决策逻辑图谱 与各模块理论逻辑脉络双向对应。  
**状态**: 已覆盖；持续深化中

---

## 🎯 **模块目标 / Module Objectives**

1. **理论逻辑脉络梳理**: 揭示各理论框架的公理体系、定理链条和推理逻辑
2. **认知规律归纳**: 总结理论发展的认知模式和规律
3. **决策逻辑建立**: 建立理论选择和算法选择的认知路径
4. **认知表征工具**: 提供思维导图、关系图、因果链等认知工具
