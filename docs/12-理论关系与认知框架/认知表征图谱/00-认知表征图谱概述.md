# 认知表征图谱概述 / Cognitive Representation Atlas Overview

## 📚 **概述 / Overview**

**文档目的**: 建立思维导图、关系图、因果链等认知表征工具的文档库，支持理论学习和应用决策。

**核心主题**:

- 思维导图：知识结构可视化
- 关系图：概念关系网络
- 因果链：推理路径分析

**主要内容**:

- 三大理论的思维导图
- 概念关系网络图
- 应用决策因果链

**适用对象**: 理论研究者、系统工程师、学习者
