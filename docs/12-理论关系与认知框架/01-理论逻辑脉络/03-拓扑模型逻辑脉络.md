## 🎯 一、公理与语义基础

### 1.1 公理层

- **拓扑空间公理**: 开集封闭性、并/交闭合
- **复形公理**: 单纯/胞腔复形的面封闭性；定向一致性
- **链复形公理**: ...→C_{i+1} →∂_{i+1} C_i →∂_i C_{i-1}→... 且 ∂_{i}∂_{i+1}=0
- **上同调公理**: δ_{i-1}δ_i=0，满足对偶性

### 1.2 语义层

- **几何语义**: 复形对应几何形状；单纯形/胞腔表示“关系的关系”
- **同调语义**: H_i(K)=ker∂_i / im∂_{i+1} 表示i维洞
- **持久语义**: 过滤 K_ε 捕获尺度变化的稳健特征

## 🔬 二、核心性质与定理链条

### 2.1 同调与上同调

- **贝蒂数β_i**: 维度=洞数量；同胚不变量
- **对偶性**: H_i 与 H^i 对偶；Poincaré对偶在流形成立

### 2.2 过滤与持久性

- **过滤序列**: K₀ ⊆ K₁ ⊆ ... ⊆ K_n
- **持久同调**: 持久条带/条形码，稳定性定理保障扰动鲁棒

### 2.3 Hodge与谱

- **离散Hodge分解**: ω = dα + δβ + γ（调和部分γ对应谐波）
- **拉普拉斯算子**: Δ_i = ∂_{i+1}∂_{i+1}ᵀ + ∂ᵀ_i∂_i；特征谱刻画结构

### 2.4 同伦与覆盖

- **同伦等价**: 保持基本群/同调；可用于简化复形
- **覆叠空间**: 万有覆叠提升到单连通；与Petri网展开类比

## 📐 三、分析方法与工具

### 3.1 复形构造

- **Vietoris-Rips / Čech**: 点云到单纯复形
- **Clique / Cell**: 从图到胞腔复形

### 3.2 持久同调

- **算法**: 边缘矩阵简化、双射配对
- **可视化**: 条形码、持久图；瓶颈距离度量差异

### 3.3 Hodge与流

- **流分解**: 将流分为梯度/旋度/谐波
- **应用**: 网络流异常检测、循环流识别

### 3.4 Mapper与形状摘要

- **Mapper**: 使用滤波函数+覆盖+聚类得到抽象图
- **应用**: 高维数据可视化、异常模式发现

### 3.5 工具链

- **GUDHI / Ripser / Dionysus**: 持久同调
- **giotto-tda**: Python流水线
- **KeplerMapper / TDAmapper**: Mapper实现
