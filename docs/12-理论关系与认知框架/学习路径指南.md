# 学习路径指南 / Learning Path Guide

## 📚 **概述 / Overview**

本文档为不同背景和需求的学习者提供系统化的学习路径，帮助快速掌握三大理论（Petri网、动态图论、拓扑模型）及其关系。

**适用对象**: 初学者、进阶学习者、应用开发者、理论研究者

---

## 🎯 **一、初学者路径 / Part 1: Beginner Path**

### 1.1 路径目标

**目标**: 快速了解三大理论的基本概念和关系，建立整体认知框架

**预计时间**: 2-3天

### 1.2 学习步骤

#### 步骤1：建立整体认知（1小时）

1. **阅读理论映射关系图谱**
   - 文档: `00-理论映射关系图谱.md`
   - 目标: 了解三大理论的基本对应关系
   - 重点: 理解库所↔顶点↔0-胞腔的对应关系

2. **查看统一理论框架思维导图**
   - 文档: `认知表征图谱/01-思维导图文档/04-统一理论框架思维导图.md`
   - 目标: 了解统一框架的整体结构
   - 重点: 理解范畴论统一框架

#### 步骤2：了解各理论基础知识（4-6小时）

1. **Petri网基础**
   - 文档: `认知表征图谱/01-思维导图文档/01-Petri网理论思维导图.md`
   - 重点: 库所、变迁、标识、点火规则
   - 时间: 1-2小时

2. **动态图论基础**
   - 文档: `认知表征图谱/01-思维导图文档/02-动态图论思维导图.md`
   - 重点: 时序网络、快照序列、图重写
   - 时间: 1-2小时

3. **拓扑模型基础**
   - 文档: `认知表征图谱/01-思维导图文档/03-拓扑模型思维导图.md`
   - 重点: 胞腔复形、链复形、同调群
   - 时间: 1-2小时

#### 步骤3：理解概念关系（2-3小时）

1. **查看概念关系网络图**
   - 文档: `认知表征图谱/02-关系图文档/01-概念关系网络图.md`
   - 目标: 理解各理论核心概念间的关系
   - 重点: 跨理论概念映射

2. **查看理论映射关系图**
   - 文档: `认知表征图谱/02-关系图文档/02-理论映射关系图.md`
   - 目标: 理解理论间的映射关系
   - 重点: 结构映射、状态映射、动态映射

### 1.3 学习成果

完成本路径后，学习者应该能够：

- ✅ 理解三大理论的基本概念
- ✅ 理解理论间的对应关系
- ✅ 理解统一框架的基本结构
- ✅ 能够识别不同理论的应用场景

## 📖 **二、深入学习路径 / Part 2: Advanced Learning Path**

**目标**: 深入理解理论的逻辑脉络、认知规律和统一性

**预计时间**: 1-2周

#### 步骤1：理解理论逻辑脉络（3-5天）

1. **Petri网理论逻辑脉络**

   - 时间: 1-2天

3. **拓扑模型逻辑脉络**

#### 步骤2：理解认知规律（2-3天）

   - 重点: 映射规则、函子定义、映射性质

2. **对偶关系总结**
   - 文档: `04-对偶关系总结/00-对偶关系总结概述.md`
   - 重点: 结构对偶、语义对偶、方法对偶
   - 时间: 1天

3. **理论融合模式**
   - 文档: `理论融合模式/00-理论融合模式概述.md`
   - 重点: 融合机制、相似性识别、统一框架
   - 时间: 0.5-1天

### 2.3 学习成果

完成本路径后，学习者应该能够：

## 💡 **七、学习建议 / Part 7: Learning Tips**

### 7.1 学习顺序建议

1. **先整体后局部**: 先了解整体框架，再深入学习细节
2. **先概念后应用**: 先理解基本概念，再学习应用方法
3. **先单一后统一**: 先理解各理论，再理解统一性

### 7.2 学习方法建议

1. **结合可视化**: 充分利用思维导图和关系图
2. **结合案例**: 通过典型场景理解理论应用
3. **结合实践**: 通过实际项目应用理论

### 7.3 学习时间建议

- **初学者路径**: 2-3天（每天2-3小时）
- **深入学习路径**: 1-2周（每天2-3小时）
- **应用实践路径**: 1周（每天2-3小时）
- **理论研究路径**: 2-3周（每天2-3小时）
