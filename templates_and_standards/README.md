## 🎯 使用指南 / Usage Guide

### 创建新文件时 / When Creating New Files

1. **参考标准文件** / **Reference Standards**:
   - 使用 `2025_2026_ULTIMATE_STANDARD.md` 作为最新标准
   - 确保文件结构符合10个必需章节要求

2. **文件命名** / **File Naming**:
   - 遵循 `NN-名称.md` 格式
   - 参考 `PROJECT_STRUCTURE.md` 中的命名规范

3. **内容结构** / **Content Structure**:
   - 参考 `CONTENT_ORGANIZATION_GUIDE.md`
   - 确保包含所有必需章节

4. **主题分类** / **Theme Classification**:
   - 参考 `THEME_HIERARCHY_MASTER.md`
   - 使用正确的主题代码格式

## 📝 必需章节清单 / Required Sections Checklist

每个知识结构文件必须包含以下10个章节：

Each knowledge structure file must include the following 10 sections:

1. ✅ **Title / 标题** (Bilingual)
2. ✅ **Table of Contents / 目录** (Auto-generated)
3. ✅ **1. Overview / 概述**
4. ✅ **2. Definition / 定义**
5. ✅ **3. Properties / 属性**
6. ✅ **4. Relations / 关系**
7. ✅ **5. Examples / 实例** (5+ examples)
8. ✅ **6. Explanations / 解释** (10 types)
9. ✅ **7. Argumentation / 论证**
10. ✅ **8. Applications / 应用**
11. ✅ **9. References / 参考文献** (包含Latest Research Frontiers)
12. ✅ **10. Status / 状态**
