## 📊 **二、属性关系矩阵模板 / Attribute Relationship Matrix Template**

### 模板结构

```markdown
    ### X.X 属性关系矩阵

    **用途**: 对比不同概念的属性特征

    | 维度 | 概念A | 概念B | 概念C | 关系类型 |
    |------|-------|-------|-------|----------|
    | **定义属性** | 属性A1<br>说明: ... | 属性B1<br>说明: ... | 属性C1<br>说明: ... | is-part-of / is-a / depends-on / dual-of / maps-to |
    | **结构属性** | 属性A2<br>说明: ... | 属性B2<br>说明: ... | 属性C2<br>说明: ... | is-part-of / is-a / depends-on / dual-of / maps-to |
    | **行为属性** | 属性A3<br>说明: ... | 属性B3<br>说明: ... | 属性C3<br>说明: ... | is-part-of / is-a / depends-on / dual-of / maps-to |
    | **关系属性** | 属性A4<br>说明: ... | 属性B4<br>说明: ... | 属性C4<br>说明: ... | is-part-of / is-a / depends-on / dual-of / maps-to |
    | **应用属性** | 属性A5<br>说明: ... | 属性B5<br>说明: ... | 属性C5<br>说明: ... | is-part-of / is-a / depends-on / dual-of / maps-to |

    **关系类型说明**:
    - **is-part-of**: 组成关系，概念A是概念B的组成部分
    - **is-a**: 继承关系，概念A是概念B的特化
    - **depends-on**: 依赖关系，概念A依赖概念B
    - **dual-of**: 对偶关系，概念A与概念B对偶
    - **maps-to**: 映射关系，概念A映射到概念B

    **关系强度**:
    - **强关系**: 概念间关系紧密，不可分离
    - **中关系**: 概念间关系中等，有一定独立性
    - **弱关系**: 概念间关系较弱，相对独立
```

### 使用说明

1. **选择对比概念**: 选择至少3个相关概念
2. **填写属性内容**: 为每个概念填写5种属性
3. **确定关系类型**: 确定概念间的关系类型
4. **说明关系强度**: 说明关系强度（强/中/弱）

## 📋 **六、模板使用指南 / Template Usage Guide**

### 6.1 模板选择

**根据对比对象选择模板**:

- 对比概念定义 → 使用概念定义矩阵模板
- 对比概念属性 → 使用属性关系矩阵模板
- 对比解释论证 → 使用解释论证矩阵模板
- 对比理论关系 → 使用理论关系矩阵模板
- 对比算法性能 → 使用算法对比矩阵模板

### 6.2 模板填写步骤

1. **确定对比对象**: 选择至少3个相关对象
2. **选择合适模板**: 根据对比内容选择模板
3. **填写矩阵内容**: 详细填写每个维度的内容
4. **添加关系说明**: 说明对象间的关系
5. **验证矩阵质量**: 检查完整性、准确性、一致性

### 6.3 质量检查清单

- [ ] 矩阵内容完整（所有维度都有内容）
- [ ] 矩阵内容准确（符合理论标准）
- [ ] 矩阵格式一致（使用统一格式）
- [ ] 关系说明明确（关系类型、强度、方向）
- [ ] 统一框架定位清晰（在统一框架中的位置）
